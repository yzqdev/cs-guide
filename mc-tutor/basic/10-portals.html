传送门与维度旅行

主世界、下界、末地——Minecraft 三大维度的通行方式与旅行技巧。

下界传送门

# 任意尺寸
- 最小内径：2×3
- 最大内径：22×22（足够让下界合金巨人通过）
- 形状可以是矩形，必须是完整的黑曜石框
- 用打火石或火焰弹激活

黑曜石获取

方法说明
水+岩浆放置水源流到岩浆源→黑曜石（最常用）
钻石镐采集自然生成的黑曜石，需要钻石镐才能挖
下界堡垒宝箱随机出现，数量有限
猪灵交易金锭换黑曜石（概率较低）

建造技巧

不要在木质基地内部启动传送门！猪灵会从下界穿过来。

# 最佳位置
- 室外空地或石质房间
- 远离重要建筑
- 周围用栅栏围起来（防止猪灵到处跑）

下界交通系统

下界坐标换算

Minecraft 的核心机制：下界 1 格 = 主世界 8 格。

# 换算公式
主世界 X = 下界 X × 8
主世界 Z = 下界 Z × 8
下界 X = 主世界 X ÷ 8
下界 Z = 主世界 Z ÷ 8

# Y 坐标不变
主世界 Y = 下界 Y

利用换算快速旅行

# 原理
在下界走 100 格 = 在主世界走 800 格
下界是主世界的「地铁系统」

# 建造步骤
1. 主世界起点 → 建传送门 A（记录坐标）
2. 进入传送门 A → 到达下界
3. 下界坐标 ÷ 8，沿目标方向走
4. 到目标位置后建传送门 B
5. 进入传送门 B → 到达主世界目标位置

# 示例：从家（0,0）到林地府邸（8000, 0）
主世界需要走 8000 格
下界只需要走 1000 格
10 分钟步行替代数小时的旅途

下界交通建设

# 下界高速公路
挖一条 3×3 的隧道 → 铺冰道
  - 用蓝冰（最滑，速度最快）
  - 配合船使用（最高可达 72 m/s）
  - 是下界最快的交通方式

# 下界铁路
动力铁轨 + 普通铁轨
  - 稳定但速度不如冰道
  - 适合长距离自动化运输

# 下界标记
在隧道墙壁上标注方向和距离
  - 用不同羊毛颜色区分方向
  - 用牌子标注主世界坐标

下界顶层交通

# 下界天花板
1. 在下界 Y=128 处找到基岩屋顶
2. 用末影珍珠卡出基岩（或使用紫颂果）
3. 到达下界顶部（基岩上方）
4. 在顶部建传送门 → 无限空地、无怪物
5. 可以建造大规模交通网络

# 注意
- 1.21+ 版本中部分卡基岩方法已被修复
- 生存模式中难度较高，建议先建立工具

末地传送门

寻找要塞

# 获取末影之眼
1. 杀烈焰人 → 获得烈焰棒
2. 烈焰棒 → 烈焰粉
3. 烈焰粉 + 末影珍珠 → 末影之眼

# 投掷末影之眼
1. 按右键投掷
2. 末影之眼会朝要塞方向飞行
3. 跟随它（注意它会碎掉）
4. 当它向下飞时，要塞就在脚下
5. 准备大量末影之眼（至少 12 个填门 + 备用导航）

激活传送门

# 要塞结构
- 末地传送门在要塞的传送门房间
- 房间里有刷怪笼（蠹虫）
- 传送门框架上有 12 个空位

# 激活步骤
1. 找到传送门房间
2. 在每个框架方块上右键放入末影之眼
3. 全部 12 个放完后传送门自动激活
4. 跳入即可到达末地

# 如果缺末影之眼
- 运气好的话部分框架已有眼睛（概率极低）
- 可以多探索几个要塞
- 或回头多收集些末影珍珠

返乡传送门（末地）

击败末影龙后，在岛屿中心出现。

# 位置
- 龙死后在折跃门处生成
- 是回到主世界或下界的唯一方式

# 特殊
- 放下后自动激活
- 不能自行建造
- 可以通过末地折跃门到达外岛

末地折跃门

击败末影龙后，在岛屿边缘生成的紫色传送门。

# 使用方式
1. 投掷末影珍珠穿过折跃门
2. 到达末地外岛
3. 外岛有大量资源：末地城、末地船、紫颂果

# 标记方向
- 投掷末影珍珠前可以先搭方块过去
- 建议在外岛建一个返乡传送门

鞘翅飞行

# 获得鞘翅
末地船（挂在船头）

# 飞行技巧
1. 跳起后按空格键滑翔
2. 俯冲增加速度
3. 拉平维持高度
4. 手持烟花火箭按右键加速
5. 落地前拉平减速，或用水桶缓冲

冰道交通

# 材料需求
- 蓝冰（速度最快）
- 船
- 对撞减速区域

# 建造
1. 挖一条 2×1 的隧道
2. 底部铺冰（1 格宽即可）
3. 放船右键上船
4. 沿冰道前进

# 速度
蓝冰: 72 m/s  ← 下界最佳
浮冰: 40 m/s
普通冰: 慢，融化快

快速参考

# 坐标换算速查
主世界 1000 格 = 下界 125 格
主世界 8000 格 = 下界 1000 格

# 各维度要点
主世界: 初始世界，资源丰富
下界:   高温，防火装备必备，猪灵需穿金装备
末地:   末影龙，大量黑曜石柱，外岛有鞘翅

# 安全提示
- 下界一定要带抗火药水
- 末地珍珠用来逃命
- 末地不要随意搭方块（可能掉入虚空）
