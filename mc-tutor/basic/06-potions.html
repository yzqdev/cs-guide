药水酿造指南

从酿造台搭建到效果药水，全面掌握 Minecraft 的药水酿造系统。

酿造基础

酿造台

# 合成
  B      B = 烈焰棒
  S      S = 圆石（3 个）
 RSSR
  S

# 放置
酿造台需要烈焰粉作为燃料（1 个烈焰粉 = 20 次酿造）

主要材料

材料来源用途
烈焰粉烈焰棒合成酿造燃料
水瓶玻璃瓶装水所有药水的基础
下界疣下界要塞将水瓶酿造成粗制药水（基础中的基础）
红石粉挖矿（Y≤15）延长持续时间
萤石粉下界采集萤石提升效果等级
火药苦力怕/宝箱制作喷溅药水
龙息末影龙的龙息攻击制作滞留药水
发酵蛛眼蛛眼+糖+棕色蘑菇合成将正面效果变负面

酿造流程概览

水瓶
  │
  ├── 下界疣 → 粗制药水（基础药水）
  │              │
  │              ├── 加效果材料 → 效果药水（3:00）
  │              │                │
  │              │                ├── +红石 → 延长版（8:00）
  │              │                ├── +萤石 → 强化版（II级）
  │              │                └── +火药 → 喷溅药水
  │              │                                    │
  │              │                                    └── +龙息 → 滞留药水
  │              │
  │              └── 发酵蛛眼 → 负面效果药水
  │
  ├── 直接加材料 → 特定药水（夜视/隐身/火抗等）
  │
  └── 发酵蛛眼 → 虚弱药水（虚弱直接用水瓶+发酵蛛眼）

药水箭

将箭与滞留药水在酿造台中合成，每瓶滞留药水可制作 8 支药水箭。

酿造实用技巧

最大化药水产出

# 配方：1 个材料 + 3 瓶粗制药水 → 3 瓶效果药水
# 酿造台一次放 3 瓶最划算

# 材料效率
- 1 烈焰棒 = 2 烈焰粉
- 1 烈焰粉 = 20 次酿造
- 1 次酿造 = 3 瓶药水
- 1 烈焰粉 = 60 瓶药水！

酿造顺序

1. 水瓶 + 下界疣       → 粗制药水 3 瓶
2. 粗制药水 + 效果材料  → 效果药水 3 瓶
3. 效果药水 + 红石粉    → 延长版（8:00）
   -或-
   效果药水 + 萤石粉    → 强化版（II 级）
4. 效果药水 + 火药      → 喷溅药水
5. 喷溅药水 + 龙息      → 滞留药水

信标与效果

信标可以为附近玩家提供持续的状态效果，无需消耗药水。

# 信标激活
1. 建造信标金字塔（铁锭/金锭/钻石/绿宝石块搭建）
2. 金字塔 3×3 → 5×5 → 7×7 → 9×9（层数越高效果越强）
3. 右键信标选择主效果
4. 满级金字塔额外开启副效果（生命恢复）

# 信标可选效果
- 速度 I / II
- 急迫 I / II
- 抗性提升 I / II
- 跳跃提升 I / II
- 力量 I / II
- 生命恢复（副效果）
