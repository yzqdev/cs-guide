Minecraft 游戏基础

游戏模式

模式说明适合场景
生存模式收集资源、合成工具、对抗怪物、维持饱食度核心玩法体验
创造模式无限资源、飞行能力、无敌状态建筑、红石实验
冒险模式不能破坏/放置方块（除非有对应工具）自定义地图
spectator模式穿墙、透视、附身生物视角观察、录制
切换模式指令：
/gamemode survival       # 生存
/gamemode creative       # 创造
/gamemode adventure      # 冒险
/gamemode spectator      # 旁观
# 简写
/gamemode 0              # 生存
/gamemode 1              # 创造
/gamemode 2              # 冒险
/gamemode 3              # 旁观

难度等级

难度怪物伤害饥饿其他
和平无怪物生成不降自动回血
简单伤害低降较慢僵尸不会破门
普通标准伤害标准默认体验
困难伤害高降得快僵尸破门、饥饿可致死
/difficulty peaceful    # 和平
/difficulty easy        # 简单
/difficulty normal      # 普通
/difficulty hard        # 困难

世界类型

类型特点
默认标准生成，各种生物群系
超平坦完全平坦，适合建筑/红石实验
巨型生物群系生物群系放大，真实感强
放大化地形起伏夸张，高山深谷
单一生物群系全图同一种群系
洞穴主世界被洞穴取代（1.18+）
调试模式显示所有方块状态

坐标系统

Minecraft 使用三维坐标 (X, Y, Z)：

  • X — 东西方向（东为正）
  • Y — 高度（0=基岩层，63=海平面）
  • Z — 南北方向（南为正）
显示坐标：F3 或 /tp ~ ~ ~
设置显示坐标：/gamerule showCoordinates true

重要高度

高度说明
Y=-64世界底部（1.18+）
Y=0基岩层顶部
Y=11传统挖钻石最佳层数
Y=63海平面
Y=255世界顶部（1.18+）
Y=319建筑上限（1.18+）

时间系统

/day    设置为白天
/night  设置为夜晚
/time set day     # 1000 tick
/time set night   # 13000 tick
/time set 0       # 黎明
/time add 1000    # 增加时间

游戏刻（Tick）

  • 1 秒 = 20 game tick
  • 1 天 = 24000 tick = 20 分钟
  • 白天: 0–12000 tick
  • 日落: 12000–13000 tick
  • 夜晚: 13000–23000 tick
  • 日出: 23000–24000 tick

初始生存流程

第一天——建立据点

1. 砍树 → 获得原木
2. 制作工作台：4个木板
3. 制作木镐 → 挖圆石 → 石镐
4. 寻找合适地点（靠山/靠水）
5. 搭建临时庇护所
6. 制作床（3个羊毛 + 3个木板）
7. 制作火把（木棍 + 木炭）

前期目标

✅ 安全庇护所
✅ 床（设置重生点）
✅ 基本工具（石镐、石斧、石剑）
✅ 火把（防止怪物生成）
✅ 食物来源（杀动物、种小麦）
✅ 熔炉（烧矿、烹饪）
✅ 箱子（存放物品）

类型群系(ID)特点
