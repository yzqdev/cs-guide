村民机制与交易

村民是 Minecraft 中最强大的资源系统——提供附魔书、钻石装备、绿宝石、经验，是一切高级玩法的经济基础。

村民基础

村民职业

村民的职业由 工作站点方块 决定。一个失业村民靠近工作站点后自动获得对应职业。

锁定交易与重刷

# 交易锁定
- 每位村民每个交易项有固定的使用次数
- 用完后需要等待补货（每天补货 2 次）
- 补货条件：村民有床且有工作站点

# 如果第一次交易不满意
- 拆除工作站点 → 村民失业 → 重新放置工作站点 → 新职业/新交易

交易机制详解

村民等级

村民有 5 个等级，交易次数越多等级越高：

等级名称所需交易次数解锁
新手新手-初始 2-4 个交易
学徒学徒10 次新交易项
熟练工熟练工20 次新交易项
专家专家30 次新交易项
大师大师40 次最终交易项

打折机制

# 打折条件
- 玩家获得「村庄英雄」效果（击败袭击）
- 玩家与村民的好感度 ≥ 15（交易 + 绿宝石块打折）

# 僵尸村民治疗
1. 用喷溅虚弱药水 + 金苹果治疗僵尸村民
2. 治疗后永久大幅打折（1 绿宝石 = 1 本附魔书！）
3. 这是获取附魔书最划算的方式

交易锁定技巧

# 保留好交易
- 某个交易很划算 → 不要最后一次交易 → 保留它
- 例如：图书管理员卖 1 绿宝石 = 1 本经验修补
- 交易到还剩 1 次后停止，等待补货

# 刷掉差交易
- 新手村民可以拆工作站重刷
- 升级后的村民不能重刷职业

村民繁殖

繁殖条件

# 村民繁殖三要素
1. 两个自愿村民（非幼年）
2. 足够的床（床数 > 村民数）
3. 食物（面包、胡萝卜、马铃薯或甜菜根）

# 食物消耗
每个村民在繁殖时需要消耗：
- 12 个面包，或
- 12 个胡萝卜，或
- 12 个马铃薯，或
- 12 个甜菜根

自动繁殖机

    [床] [床] [床] [床]   ← 3 格高房间
      ↑ 村民跳起来丢食物
    [农民]      [农民]    ← 2 格高房间，丢食物给村民
      ↑ 收集农作物
      [农田]

最简单的自动繁殖机：农民种地 → 收菜 → 丢给其他村民 → 村民捡起后繁殖。

村民运输

# 用矿车运输（最安全）
1. 放置铁轨 + 矿车
2. 右键村民上矿车
3. 推到目标位置

# 用水流运输
1. 挖一条 1×1 水渠
2. 村民掉入后自动顺流移动

# 用船运输（水域效率最高）
1. 在陆地上放船
2. 村民会自动上船

交易大厅工程

将多个村民集中管理，最大化利用交易系统。

简单交易大厅设计

# 单人小档设计
房间布局：
  [讲台] [村民] [栅栏/玻璃] → 玩家站在外面交易

每个村民一格空间，前面放工作站点方块
玩家在外面能看到村民并交易

# 推荐配置（8 位村民）
1. 图书管理员 × 3   → 经验修补、锋利 V、效率 V 附魔书
2. 盔甲匠 × 2       → 全套钻石护甲
3. 工具匠 × 1       → 钻石镐/钻石斧
4. 武器匠 × 1       → 钻石剑
5. 牧师 × 1         → 末影珍珠、红石

僵尸村民打折机

# 原理
1. 将村民僵尸化
2. 再治疗（喷溅虚弱药水 + 金苹果）
3. 治疗后永久大幅打折
4. 可以反复操作直到获得极低的交易价格

# 关键
- 困难模式下僵尸转化率 100%
- 简单/普通模式下概率较低
- 注意保护好村民，防止被完全杀死

铁傀儡机制

生成条件

# 村民生成铁傀儡
- 至少有 3 个村民
- 至少 75% 的村民工作过（最近一天内）
- 村民必须能看到铁傀儡生成位置

# 自然生成
- 村庄范围内有 ≥10 个村民 + ≥21 个门（旧版本）
- 新版本：村民恐慌时概率生成

铁农场

利用村民生成铁傀儡来获取铁锭：

# 核心设计思路
1. 至少 3 个村民（带床和工作站）
2. 村民上方有生成平台
3. 铁傀儡生成后 → 水流收集 → 击杀

# 产出效率
- 简单设计：约 40 铁/小时
- 优化设计：约 200+ 铁/小时

袭击机制

触发袭击

# 获得不祥之兆
- 击杀袭击队队长（头顶灾厄旗帜的掠夺兽/卫道士）
- 获得不祥之兆效果

# 进入村庄
- 带不祥之兆效果进入有村民的村庄
- 触发袭击（头顶出现血条进度）

袭击奖励

波数敌人强度主要奖励
1-3掠夺兽、卫道士绿宝石、箭
4-5唤魔者加入绿宝石、不死图腾
6-7女巫药水、红石
胜利-村庄英雄 效果（村民打折）

村庄英雄期间（40分钟）所有村民交易大幅打折，是大量囤货的最佳时机。

快速参考

# 村民繁殖材料
面包：1 小麦 → 6 面包（每 12 面包繁殖一次）
胡萝卜/马铃薯：9 个繁殖一次

# 首选图书管理员交易
经验修补 → 必需品（附魔到所有工具上）
锋利 V → 武器附魔
效率 V → 挖掘附魔
保护 IV → 护甲附魔

# 村民工程三原则
1. 给每个村民配床
2. 给每个村民配工作站点
3. 村民与僵尸要隔开保护
