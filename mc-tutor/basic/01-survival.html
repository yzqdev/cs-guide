生存模式完整攻略

从第一天落地到击败末影龙，覆盖生存模式的完整流程与核心技巧。

第一天：活下去

你的第一个 Minecraft 日只有 10 分钟（现实时间），合理利用每一秒。

落地必修课

1. 砍树（4-6 块原木）
2. 合成工作台
3. 合成木镐 → 挖 17 块圆石 → 石镐/石剑/石斧
4. 杀动物获取食物（牛、羊、鸡、猪）
5. 收集羊毛（杀 3 只羊，或者用线合成）
6. 造床（3 木板 + 3 羊毛）
7. 天黑前找/造庇护所

第一夜做什么

活动说明
挖矿往下挖到 Y=11 附近，进入矿道
扩充基地挖出一个房间，放工作台、熔炉、箱子
烧制将圆石烧成石头、铁原矿烧成铁锭
探索洞穴如果有信心，可以举着火把探索附近的天然洞穴

紧急庇护所

造不了床？挖一个 1×2 的洞，把自己埋起来，等天亮。

前三天：建立基地

寻找合适地点

理想的基地位置：

  • 靠近水源（河流、海洋、或大片平地）
  • 附近有树木
  • 地势平坦或容易整平
  • 附近有洞穴入口（方便白天挖矿）

优先收集

必需：
  铁锭 (iron_ingot) ≥ 24  → 铁套 + 铁镐(iron_pickaxe) + 铁剑(iron_sword) + 水桶(bucket) + 盾牌(shield)
  食物 ≥ 20    → 牛排(cooked_beef)/猪排(cooked_porkchop)（烤熟后回饱食度高）
  煤炭 (coal) ≥ 16    → 火把(torch)（点亮洞穴和基地）
  羊毛 (white_wool) ≥ 3     → 床(red_bed)（重置重生点）

推荐：
  黑曜石 (obsidian) ≥ 10  → 下界传送门(nether_portal)（用水桶+钻石镐或岩浆+水造）
  钻石 (diamond) ≥ 3     → 钻石镐(diamond_pickaxe)（挖黑曜石必需）
  皮革 (leather) ≥ 24    → 书(book)（附魔用）

第一组铁锭做什么

优先级物品ID用量理由
⭐⭐⭐铁镐iron_pickaxe3挖钻石必须铁镐以上
⭐⭐⭐盾牌shield1+6木板格挡弓箭和爆炸
⭐⭐⭐铁剑iron_sword2比石剑高很多伤害
⭐⭐铁胸甲iron_chestplate8提供最多护甲的部件
⭐⭐水桶bucket3灭火、爬墙、刷黑曜石
⭐铁锹/铁斧iron_shovel / iron_axe1+1挖沙土、砍树加速
⭐剪刀shears2收集羊毛不用杀羊

早期目标（第 3-7 天）

找到钻石

# 钻石在哪里？
- 只在 Y=-64 ~ 16 之间生成
- 最集中的层数：Y=-59（1.21 版本起）
- 1.21 前传统挖矿层：Y=11

# 最佳挖矿策略
1. 鱼骨挖矿法：主巷道在每个 Y 坐标 + 分支道隔 3 格
2. 天然洞穴探索：效率最高，但要小心怪物
3. 需要至少铁镐（钻石镐才能挖到钻石！）
4. 带够火把、食物、备用工具

打造附魔装备

# 经验来源
- 杀怪（推荐找刷怪笼做简易经验农场）
- 挖矿（烧矿给的 exp 很少）
- 交易（与村民换绿宝石有 exp）
- 钓鱼（慢）

# 附魔台要求
- 书架 15 个（围绕附魔台一圈留一格间距）
- 附魔台优先附魔：钻石镐（效率）→ 铁剑（锋利）→ 钻石胸甲（保护）

建立简易农场

  • 小麦/胡萝卜/马铃薯 (wheat/carrot/potato): 水源 + 耕地 + 光照
  • 甘蔗 (sugar_cane): 水边种植，用于做书(book)（附魔）和糖(sugar)
  • 动物: 用小麦(wheat)（牛/羊）或胡萝卜(carrot)（猪）繁殖
  • 树场: 各种树苗种一片，持续供木

中期目标（第 7-14 天）

进入下界

# 建造下界传送门 (nether_portal)
1. 用黑曜石(obsidian)搭出至少 4×5 的框架（外框）
2. 用打火石(flint_and_steel)（铁锭+燧石）激活
3. 注意：不要在基地内部激活，猪灵(piglin)会过来

# 下界必做清单
1. 找下界要塞(nether_fortress)（目标：烈焰棒(blaze_rod)）
2. 收集烈焰棒 → 烈焰粉(blaze_powder) → 末影之眼(eye_of_ender)
3. 寻找堡垒遗迹(bastion_remnant)（目标：下界合金碎片(netherite_scrap)）
4. 挖远古残骸(ancient_debris)（Y=15 附近，需要用钻石镐）

下界合金装备

# 升级路线
1. 挖远古残骸(ancient_debris) → 烧成下界合金碎片(netherite_scrap)
2. 4 碎片 + 4 金锭(gold_ingot) → 下界合金锭(netherite_ingot)
3. 锻造模板(netherite_upgrade_smithing_template) + 下界合金锭 + 钻石装备 → 下界合金装备(netherite_xxx)

# 重点升级
优先级：镐子(pickaxe) → 剑(sword) → 胸甲(chestplate) → 头盔(helmet) → 靴子(boots) → 裤子(leggings)
- 下界合金装备：击退抗性、不会烧毁
- 强烈建议至少先升级镐子和剑

村民交易大厅

建立一个小型村民交易点（详见 村民机制）:

  • 村民繁殖 → 多职业搭配
  • 锁定有利交易（打折后不要交易到锁）
  • 图书管理员卖附魔书（最好获取附魔的方式）

后期目标（第 14 天+）

前往末地

# 准备清单
1. 末影之眼(eye_of_ender) ≥ 18 个（烈焰粉(blaze_powder) + 末影珍珠(ender_pearl)合成）
2. 投掷末影之眼 → 跟随 → 找到要塞(stronghold)
3. 要塞里找到传送门房间
4. 用末影之眼填充传送门框架(end_portal_frame)
5. 跳入传送门（做好准备后再跳）

# 战前装备
- 下界合金剑(netherite_sword)（锋利 V 以上）
- 弓(bow)（力量 V，无限或大量箭(arrow)）
- 末影珍珠(ender_pearl)（躲龙息、上塔）
- 水桶(water_bucket)（落地缓冲）
- 石/铁镐（挖末地石(end_stone)）
- 金苹果(golden_apple)、附金苹果(enchanted_golden_apple)
- 缓降药水(potion_of_slow_falling)（可选）

末影龙战斗攻略

# 第一件事：摧毁水晶
- 末地黑曜石柱上有末影水晶
- 水晶会给龙回血
- 用弓箭射掉，或者爬上去敲掉
- 推荐带脚手架或末影珍珠

# 战斗阶段
1. 龙在岛屿上空盘旋
2. 摧毁所有水晶后龙不再回血
3. 龙会俯冲攻击，盾牌可格挡
4. 等龙停在折跃门上时输出

# 核心技巧
- 不要看末影人的眼睛（戴南瓜头）
- 有水桶落地防摔死
- 缓慢药水箭对龙有效

战利品与奖励

战利品ID用途
龙蛋dragon_egg装饰（可以拾取，会传送）
大量经验—附魔到顶级
折跃门end_gateway前往末地外岛
鞘翅elytra在末地船中获得，飞行神器
潜影壳shulker_shell合成潜影盒(shulker_box)（移动箱子）

通关后做什么

击败末影龙只是开始，Minecraft 的终局内容：

探索末地外岛

  • 通过折跃门到达末地外岛
  • 末地船：获得鞘翅(elytra)、龙头(dragon_head)
  • 末地城(end_city)：大量战利品、潜影贝(shulker)
  • 紫颂果(chorus_fruit)：食物 + 传送

建造大规模工程

项目说明
守卫者农场 (guardian_farm)最快经验的来源之一
猪灵金农场 (piglin_gold_farm)大量金锭 + 经验
村民交易所全附魔书 + 钻石装备
分类存储系统自动分类所有物品
冰道交通网 (blue_ice_road)连接各大生物群系
凋灵骷髅农场 (wither_skeleton_farm)下界之星(nether_star) → 信标(beacon)

挑战成就

# 隐藏成就
- "风筝"：用附魔弓射中潜影贝
- "热门的景点"：访问所有生物群系
- "探索的时光"：触发所有生物群系
- "均衡饮食"：吃下每一种食物
- "成双成对"：繁殖每种动物
- "狂乱的鸡尾酒"：同时拥有所有药水效果
- "为什么会变成这样呢"：同时拥有所有负面效果

生存核心技巧汇总

通用法则

法则说明
不要垂直往下挖下面可能是熔岩或洞穴，掉进去必死
随身带水桶灭火、爬墙、落地缓冲、刷黑曜石
天黑前回基地晚上在外游荡死亡率极高
火把每隔 6 格放一个防止怪物在视野盲区生成
不裸矿挖矿至少铁甲再下矿，小心岩浆
保持饱食度饱食度少于 18 不会自然回血
备份存档重要建筑前 F2 截图 + 备份世界

死亡不掉落模组

如果你在服务器上或开启了 keepInventory：

/gamerule keepInventory true

快捷操作

F3  → 调试信息（坐标、帧率、群系）
F2  → 截图
F3+Q  → 显示所有 F3 快捷键列表
F3+B  → 显示实体碰撞箱
F3+G  → 显示区块边界
