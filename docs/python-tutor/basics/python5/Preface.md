# 前言 #

通常都听到别人说，计算机很牛逼，很聪明，其实计算机一点都不聪明，光是你要跟他沟通，都会气 shi 你，聪明的是在写程序的你。

写程序就是跟计算机沟通，告诉它要做什么。

竟然是这样，那么肯定缺少不了一些沟通逻辑。比如你要告诉计算机在什么情况下做什么？或者在哪个时间点做什么？

这都需要用到逻辑判断。这一章节，主要就是说这个。
