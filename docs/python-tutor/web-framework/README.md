# Python Web 框架

> Python 拥有丰富的 Web 框架生态，从全栈到轻量级、从同步到异步，总有一款适合你。

<Catalog />

## 框架对比

| 框架 | 类型 | 特点 | 适用场景 |
|------|------|------|---------|
| **Django** | 全栈框架 | 大而全、ORM、Admin 面板、生态成熟 | 大型项目、CMS、内容平台 |
| **Flask** | 轻量微框架 | 灵活、可扩展、蓝本支持 | 小型项目、API、原型开发 |
| **FastAPI** | 异步高性能 | 自动文档、类型校验、async/await | API 服务、高并发、ML 部署 |
| **Sanic** | 异步框架 | 高性能、类 Flask 风格 | 高吞吐量 API、实时服务 |
