# Visual Studio 配置

## 技巧

## 配置格式化

[配置文件仓库](https://github.com/yzqdev/my-configs.git)

格式化快捷键：`Ctrl+K, Ctrl+F`

要在调试停止时自动关闭控制台，请启用"工具" → "选项" → "调试" → "调试停止时自动关闭控制台"。

## 常用配置

- 配置 `.editorconfig` 和 `.gitignore`
- 配置自动补全、搜索：删除字符后显示完整列表，并打上勾
