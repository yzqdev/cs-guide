# Windows 快捷键大全

> 掌握这些快捷键，让你的 Windows 操作效率翻倍。

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + C` | 复制 |
| `Ctrl + X` | 剪切 |
| `Ctrl + V` | 粘贴 |

| `Ctrl + Y` | 重做 |
| `Ctrl + A` | 全选 |
| `Ctrl + S` | 保存 |
| `Ctrl + F` | 查找 |
| `Ctrl + H` | 替换 |
| `Ctrl + P` | 打印 |
| `Ctrl + Shift + Esc` | 打开任务管理器 |
| `Alt + F4` | 关闭当前窗口 |
| `Alt + Tab` | 切换窗口 |
| `Alt + Space` | 打开窗口菜单（移动/大小/最小化/关闭） |
| `F2` | 重命名 |
| `F5` | 刷新 |

## 剪贴板增强

Win10/11 内置了剪贴板历史功能：

1. 按 `Win + V` 打开剪贴板历史
2. 可以查看最近复制的多条内容（文本、图片）
3. 点击任意条目即可粘贴
4. 可以固定常用条目

> 需要在 **设置 → 系统 → 剪贴板** 中开启"剪贴板历史记录"。

## 虚拟桌面（多桌面）

虚拟桌面适合将不同任务分组：

- `Win + Ctrl + D` — 新建桌面
- `Win + Ctrl + ←/→` — 切换桌面
- `Win + Ctrl + F4` — 关闭当前桌面
- `Win + Tab` — 查看所有桌面和窗口

**使用场景：**
- 桌面1：工作（VS Code + 浏览器 + 终端）
- 桌面2：聊天通讯（微信 + 钉钉）
- 桌面3：娱乐（视频 / 音乐）
