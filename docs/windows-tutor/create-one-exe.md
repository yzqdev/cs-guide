# 开发单个exe桌面应用程序

:::tip
第一类：C#或者VB，开发框架WPF、WinForm。界面可以非常绚丽，功能强大，也很成熟！开发上手难度相对较小！  
第二类：QT：C++,界面不错，性能好，跨平台，问题在于使用难度大，但收益也特别好，属于高风险高回报的那一类。另外开发环境比之上面俩略差。  
第三类：Web开发，使用Electron，使用语言html,js,css，功能强大，界面绚丽，跨平台，开发成本低，可以说是集优点于一身，除了性能是真的差。

:::
