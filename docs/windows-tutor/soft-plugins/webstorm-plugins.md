# WebStorm 插件推荐

以下是适用于 WebStorm 的前端开发常用插件。

## 环境配置类

### .env Files Support
- **功能**: 支持 `.env` 环境变量文件的语法高亮和补全
- **适用**: 使用环境变量配置的项目

## 代码格式化类

### Prettier
- **功能**: 统一代码格式化风格，支持多种语言
- **特点**: 团队协作时保持代码风格一致
- **配合**: 建议安装 Prettier ESLint 插件实现自动修复

## 推荐的其他插件

参考 [公共插件](common-plugins.md) 获取更多适用于 WebStorm 的插件推荐，包括：

- **Vue.js 相关**: Vue.js DevTools、Vue Template Support
- **React 相关**: React Native Tools
- **代码质量**: ESLint、Stylelint
- **效率工具**: Key Promoter X、AceJump
