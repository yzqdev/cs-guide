# Tampermonkey 油猴脚本推荐

以下是一些实用的 Tampermonkey 脚本，可大幅提升网页浏览体验。

## 通用增强类

### 自动无缝翻页
- **功能**: 自动加载下一页内容，实现无缝阅读体验
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/419215-%E8%87%AA%E5%8A%A8%E6%97%A0%E7%BC%9D%E7%BF%BB%E9%A1%B5)

### HTML5 视频播放器增强
- **功能**: 增强网页视频播放器功能，支持倍速、画质切换等
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/381682-html5%E8%A7%86%E9%A2%91%E6%92%AD%E6%94%BE%E5%99%A8%E5%A2%9E%E5%BC%BA%E8%84%9A%E6%9C%AC)

## 网站美化类

### 网易云扁平风格样式
- **功能**: 将网易云音乐网页版改为扁平风格，更简洁美观
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/369688-%E9%80%82%E7%94%A8%E4%BA%8E%E7%BD%91%E6%98%93%E4%BA%91%E9%9F%B3%E4%B9%90%E6%89%81%E5%B9%B3%E9%A3%8E%E6%A0%BC%E6%A0%B7%E5%BC%8F%E8%A1%A8%E7%9A%84%E8%BE%85%E5%8A%A9%E7%94%A8%E6%88%B7%E8%84%9A%E6%9C%AC)

### 知乎美化
- **功能**: 优化知乎页面样式，提升阅读体验
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/402808-%E7%9F%A5%E4%B9%8E%E7%BE%8E%E5%8C%96)

### 知乎增强
- **功能**: 增强知乎功能，添加更多实用特性
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/419081-%E7%9F%A5%E4%B9%8E%E5%A2%9E%E5%BC%BA)

### GitHub 增强
- **功能**: 增强 GitHub 功能，支持快速下载等
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/412245-github-%E5%A2%9E%E5%BC%BA-%E9%AB%98%E9%80%9F%E4%B8%8B%E8%BD%BD)

## 内容过滤类

### 药方（Yet Another Weibo Filter）
- **功能**: 微博内容过滤器，可自定义屏蔽关键词、用户等
- **安装**: [Greasy Fork](https://greasyfork.org/zh-CN/scripts/3249-yet-another-weibo-filter)

## 视频网站增强

### Bilibili Evolved
- **功能**: 强大的哔哩哔哩增强脚本，支持自动播放、跳过片头、画质切换等
- **特点**: 功能非常全面的 B 站增强工具
- **安装**: [GitHub](https://github.com/the1812/Bilibili-Evolved)
