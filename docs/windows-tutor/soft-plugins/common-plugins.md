# 公共插件

以下是适用于 JetBrains 系列 IDE（如 IntelliJ IDEA、WebStorm、GoLand 等）的常用插件推荐。

## 效率提升类

### Key Promoter X
- **功能**: 当您使用鼠标操作时，提示相应的快捷键，帮助您养成使用快捷键的习惯
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/9792-key-promoter-x)

### AceJump
- **功能**: 快速跳转到文档中的任意位置，通过输入字符快速定位
- **特点**: 极大提升光标定位效率

### CamelCase
- **功能**: 快速切换变量命名格式（驼峰式、下划线式、连字符式等）

### String Manipulation
- **功能**: 提供多种字符串操作工具（反转、大小写转换、排序等）

## 代码质量类

### Alibaba Java Coding Guidelines
- **功能**: 阿里巴巴 Java 代码规范检查插件，帮助团队遵循统一的编码标准
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/10046-alibaba-java-coding-guidelines)

### Easy Javadoc
- **功能**: 自动生成 JavaDoc 注释，支持多种注释模板
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/12977-easy-javadoc)

## API 开发类

### Restful Fast Request
- **功能**: RESTful API 快速测试工具，支持请求构建、响应解析、环境管理
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/16988-restful-fast-request)

### JSON Parser
- **功能**: JSON 数据格式化和解析工具

## 视觉美化类

### Indent Rainbow
- **功能**: 为代码缩进添加彩虹颜色，便于区分代码层级
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/13308-indent-rainbow)

### Rainbow Brackets
- **功能**: 为括号添加彩色配对显示，便于识别代码块
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/10080-rainbow-brackets)

### Atom Material Icons
- **功能**: 提供精美的 Material Design 风格图标

### Atom One Dark Theme / One Dark Native / One Dark Theme
- **功能**: 流行的深色主题，护眼且美观

### CodeGlance
- **功能**: 在编辑器右侧显示代码缩略图，便于快速导航

## AI 辅助类

### GitHub Copilot
- **功能**: AI 代码补全助手，基于机器学习的智能代码建议
- **链接**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/17718-github-copilot)

## 版本控制类

### GitToolBox
- **功能**: 增强 Git 功能，提供更丰富的版本控制体验

### Gitmoji / Gitmoji Plus
- **功能**: 提交时快速选择 emoji 标签，使提交信息更直观

### Gitee
- **功能**: 集成 Gitee 代码托管平台
