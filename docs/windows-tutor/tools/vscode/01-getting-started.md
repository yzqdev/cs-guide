# 01 — 安装与入门

> 本章带你从零开始安装 VS Code，熟悉界面布局，掌握最基础的操作。

---

## 1.1 版本选择

VS Code 有三个版本：

| 版本                   | 说明                                 | 适用场景           |
| ---------------------- | ------------------------------------ | ------------------ |
| **Stable（稳定版）**   | 每月更新，经过充分测试               | 日常开发，推荐     |
| **Insiders（内测版）** | 每日构建，包含最新特性，偶尔有 bug   | 尝鲜、测试新功能   |
| **Portable（便携版）** | 解压即用，配置和数据保存在安装目录内 | 绿色软件、U 盘携带 |

### 下载地址

- **Stable / Insiders**: [https://code.visualstudio.com/](https://code.visualstudio.com/)
- **Portable 版**: 下载 `.zip` 包，解压后在根目录创建 `data` 文件夹即启用便携模式

---

## 1.2 安装后必做三件事

### ① 安装 `code` 命令

在终端中直接用 `code .` 打开当前目录：

1. 打开 VS Code
2. 按 `Ctrl+Shift+P` 打开命令面板
3. 输入 `Shell Command: Install 'code' command in PATH`
4. 回车确认

验证：

```bash
# 在任意终端中运行
code --version

# 打开当前目录
code .
```

### ② 开启 Settings Sync

登录账号同步设置和扩展（详见 [09 工作区与同步](./09-workspace-profile-sync.md)）：

1. 点击左下角齿轮图标 → `Turn on Settings Sync...`
2. 选择 GitHub 或 Microsoft 账号登录
3. 选择要同步的内容（推荐全选）

### ③ 安装语言支持

VS Code 默认只对纯文本、Markdown 等少数格式有良好支持。打开侧边栏扩展视图（`Ctrl+Shift+X`），搜索并安装你需要的语言包：

- **中文界面**：搜索 `Chinese (Simplified)` 安装后重启
- **Python**：搜索 `Python`（微软官方）
- **JavaScript/TypeScript**：内置支持，无需额外安装
- **Java**：搜索 `Extension Pack for Java`
- **Go**：搜索 `Go`（官方）

## 1.3 界面布局

```
┌──────────────────────────────────────────────────────────┐
│  活动栏  │   侧边栏     │        编辑器区域           │
│ (Activity │   (Side Bar) │       (Editor Group)        │
│  Bar)     │              │                             │
│           │              │  ┌─────┬──────┐            │
│ ● 文件    │  文件树      │  │Tab1 │ Tab2 │  面板     │
│ ● 搜索    │  搜索面板    │  ├─────┴──────┤  (Panel)   │
│ ● 源码    │  Git 面板    │  │  编辑器    │ ──────── │
│ ● 调试    │  调试面板    │  │  内容      │ 终端      │
│ ● 扩展    │  扩展面板    │  └────────────┘ 输出      │
│ ● 远程    │             │                │ 问题      │
└──────────────────────────────────────────────────────────┘
```

### 各区域说明

| 区域           | 快捷键       | 作用                                 |
| -------------- | ------------ | ------------------------------------ |
| **活动栏**     | —            | 切换侧边栏的不同视图                 |
| **侧边栏**     | `Ctrl+B`     | 切换显示/隐藏                        |
| **编辑器区域** | `Ctrl+\`     | 分屏，支持多标签                     |
| **面板**       | `` Ctrl+` `` | 显示终端、输出、问题、调试控制台     |
| **状态栏**     | —            | 底部信息栏（行号、编码、Git 分支等） |

### 窗口管理

- **命令面板**: `Ctrl+Shift+P` — VS Code 的"任意门"，所有功能都能从这里触发
- **快速打开文件**: `Ctrl+P` — 输入文件名模糊搜索
- **编辑器分屏**: `Ctrl+\` — 左右分屏，拖拽 Tab 可调整布局
- **切换全屏**: `F11`
- **禅模式**（Zen Mode）: `Ctrl+K Z` — 隐藏所有 UI，只留编辑器，专注写作

## 1.4 第一个文件

1. 按 `Ctrl+N` 新建文件
2. 按 `Ctrl+S` 保存为 `hello.py`
3. 输入以下代码：

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("VS Code"))
```

4. 按 `Ctrl+`` 打开终端，运行：

```bash
python hello.py
```

5. 输出：`Hello, VS Code!`

## 1.5 打开项目的方式

```bash
# 在终端中打开
code .                    # 打开当前目录
code /path/to/project     # 打开指定目录
code -r .                 # 在当前窗口打开（不新建窗口）
code -n .                 # 强制新建窗口打开

# 对比文件
code --diff file1.js file2.js

# 等待文件关闭后再返回终端
code --wait edit.md
```

也可以在 VS Code 中：

- **File > Open Folder...** — 打开文件夹
- **File > Open File...** — 打开单个文件
- **File > Add Folder to Workspace...** — 添加多个根目录到同一工作区
