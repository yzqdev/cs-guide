## 10.1 高频快捷键（必记）

这 15 个快捷键覆盖了日常 80% 的操作：

| 快捷键                    | 功能                  | 记忆技巧           |
| ------------------------- | --------------------- | ------------------ |
| `Ctrl+Shift+P` / `F1`     | 命令面板              | VS Code 的"任意门" |
| `Ctrl+P`                  | 快速打开文件          | **P** → Pick       |
| `` Ctrl+` ``              | 打开终端              | 反引号像终端图标   |
| `Ctrl+B`                  | 切换侧边栏            | **B** → Bar        |
| `Ctrl+\`                  | 分屏                  | 反斜杠像分割线     |
| `Ctrl+J`                  | 切换面板（终端/输出） |                    |
| `Ctrl+W`                  | 关闭当前 Tab          |                    |
| `Ctrl+S`                  | 保存                  |                    |
| `Ctrl+Z` / `Ctrl+Shift+Z` | 撤销 / 重做           |                    |
| `Ctrl+D`                  | 选中下一个相同的词    | 多次按可多选       |
| `Ctrl+Shift+L`            | 选中所有相同词        | 批量替换           |
| `Alt+↑/↓`                 | 移动行                | 最常用的编辑操作   |
| `Shift+Alt+F`             | 格式化文档            |                    |
| `F12`                     | 跳转到定义            | 读代码的神键       |

## 10.14 自定义快捷键

如果默认快捷键不合手，可以随时修改：

```jsonc
// keybindings.json
// 打开方式：命令面板 → Preferences: Open Keyboard Shortcuts (JSON)

[
  // 将 Ctrl+Y 改为"删除行"（红框习惯）
  {
    "key": "ctrl+y",
    "command": "editor.action.deleteLines",
    "when": "editorTextFocus",
  },
  // 将 Ctrl+Shift+/ 改为"切换块注释"
  {
    "key": "ctrl+shift+/",
    "command": "editor.action.blockComment",
    "when": "editorTextFocus && !editorReadonly",
  },
  // 给终端清屏绑定快捷键
  {
    "key": "ctrl+q",
    "command": "workbench.action.terminal.clear",
  },
  // 给命令面板绑定更顺手的键
  {
    "key": "ctrl+e",
    "command": "workbench.action.showCommands",
    "when": "editorTextFocus",
  },
]
```
