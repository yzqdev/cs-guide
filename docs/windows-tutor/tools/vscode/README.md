# VS Code 完全教程

> 工欲善其事，必先利其器。VS Code 是现代开发者最强大的编辑器之一，它不只是编辑器，更是一个可无限扩展的开放平台。
>
> 本教程分为 **使用篇** 和 **扩展开发篇**，从入门到精通，帮你真正用好 VS Code。
