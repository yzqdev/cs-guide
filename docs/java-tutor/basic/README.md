## 学习建议

1. **按顺序学习**：从 Part 1 开始，打好基础再进入 Part 2、3、4
2. **动手练习**：每个文件末尾有练习，务必亲手写一遍
3. **配合 java-tips**：学完基础后，参考 [java-tips](../java-tips/README.md) 中的实用技巧
4. **查阅设计模式**：了解基础后，参考 [设计模式](../design-pattern/README.md) 提升架构能力
