2、CSS 优先级的选择过程？

优先级复合就近原则，同权重的情况下有限选择最近的属性。

载入样式的话是以最后载入的定位为准。

优先级：

!important > id > class > tag（important要优先于内联样式）

9、为什么要初始化CSS样式？

因为浏览器的兼容性问题，不同的浏览器对某些标签有自己的默认值，如果没有实现css的初始化，也就是没有解决浏览器的兼容性问题，那么会造成页面在不同浏览器下显示差异的问题。
