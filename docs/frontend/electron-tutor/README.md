# electron学习笔记

::tip
使用 JavaScript，HTML 和 CSS 构建跨平台的桌面应用程序
:::

## 快速开始

```shell
git clone https://github.com/electron/electron-quick-start

cd electron-quick-start
npm install && npm start
```

[文档地址](https://www.electronjs.org/docs/latest/)
