# 好用的 Chrome 插件推荐

## 🔧 前端开发

| 插件 | 说明 |
|---|---|
| **[React Developer Tools](https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi)** | React 组件调试（F12 → Components/Profiler） |
| **[Vue.js DevTools](https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd)** | Vue 组件调试，支持 Vue 2/3 |
| **[Redux DevTools](https://chrome.google.com/webstore/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd)** | Redux 状态管理调试 |
| **[FeHelper](https://www.baidufei.com/fehelper/)** | 前端助手：JSON 格式化、编码解码、正则测试、页面取色等 |
| **[Wappalyzer](https://www.wappalyzer.com/)** | 查看网站使用的技术栈（框架、CMS、分析工具等） |
| **[JSON Formatter](https://github.com/callumlocke/json-formatter)** | 浏览器中 JSON 自动格式化显示 |
| **[XPath Helper](https://chrome.google.com/webstore/detail/xpath-helper/hgimnogjllphhhkhlmebbmlgjoejdpjl)** | XPath 提取工具，方便爬虫开发 |
| **[WhatFont](https://chrome.google.com/webstore/detail/whatfont/jabopobgcpjmedljpbcaablpmlkfcidd)** | 查看网页使用的字体 |
