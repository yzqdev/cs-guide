## 简介

Angular 是一个基于 TypeScript 的现代 Web 应用框架，由 Google 维护。与 React 和 Vue 不同，Angular 是一个完整的框架（而非库），内置了路由、HTTP 客户端、表单处理、依赖注入等开箱即用的功能。
