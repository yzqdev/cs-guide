## 简介

React 是由 Facebook（Meta）开发和维护的用于构建用户界面的 JavaScript 库。React 使用**虚拟 DOM** 和**组件化**开发模式，以其声明式、高效和灵活的特性成为前端开发的主流选择。

### 核心特性

- ⚛️ **声明式** - 以声明方式描述 UI，状态变化时自动更新
- 🧩 **组件化** - 将 UI 拆分为独立、可复用的组件
- 🔄 **虚拟 DOM** - 通过 diff 算法最小化真实 DOM 操作
- ⏱ **Fiber 架构** - 可中断的异步渲染（React 18+）
