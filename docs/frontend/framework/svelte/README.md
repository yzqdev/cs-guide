## 简介

Svelte 是一个创新的前端框架，与 React、Vue 不同，它**没有虚拟 DOM**，而是在**编译阶段**将组件转换为高效的 JavaScript 代码，直接操作 DOM。这使得 Svelte 应用体积更小、性能更好。

### 核心特性

- 🚫 **无虚拟 DOM** - 编译时优化，运行时最小化
- 🎯 **真正的响应式** - 声明式变量赋值即可触发更新
- 📦 **极小的打包体积** - 编译后的代码非常精简
- 🧩 **内置状态管理** - 无需额外库，Store 原生支持
- 🎨 **CSS 默认作用域** - 组件样式自动隔离
- 🌀 **动画内置** - 原生动画和过渡系统
