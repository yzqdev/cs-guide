## 简介

SolidJS 是一个声明式、高效且轻量级的 JavaScript 库，用于构建用户界面。它采用与 React 类似的 JSX 语法，但使用真实的 DOM（而非虚拟 DOM），通过细粒度的响应式系统实现极致性能。

### 核心特性

- ⚡ **极致性能** - 没有虚拟 DOM，直接操作真实 DOM，编译时优化
- 🎯 **细粒度响应式** - 基于 Signal 的响应式系统，只更新变化的部分
- 🔧 **JSX 语法** - 与 React 类似的开发体验
- 📦 **轻量级** - 核心库仅约 7KB（gzip）
