# 富文本编辑器

## 推荐列表

| 编辑器 | 特点 | 框架支持 |
|--------|------|:--------:|
| [WangEditor](https://www.wangeditor.com/) | 轻量，中文友好，Vue/React 都支持 | Vue 2/3, React |
| [TinyMCE](https://www.tinymce.com/) | 功能强大，插件丰富，老牌编辑器 | Vue, React, 原生 |
| [CKEditor](https://ckeditor.com/) | 企业级，协作编辑，付费功能 | Vue, React, 原生 |
| [Quill](https://quilljs.com/) | 现代，API 简洁，可扩展 | 原生，社区封装 |
| [MavonEditor](https://www.npmjs.com/package/mavon-editor) | Markdown 编辑器，Vue 友好 | Vue 2/3 |
| [md-editor-v3](https://imzbf.github.io/md-editor-v3/) | 轻量 Markdown 编辑器，Vue 3 | Vue 3 |
| [Monaco Editor](https://microsoft.github.io/monaco-editor/) | VS Code 内核，代码编辑器 | 原生 |

## Vue 3 推荐

### Markdown 编辑器

- [md-editor-v3](https://imzbf.github.io/md-editor-v3/) - 轻量级，Vue 3 原生支持
- [bytemd](https://github.com/bytedance/bytemd) - 字节跳动出品，基于 Svelte 构建

### 富文本编辑器

- [WangEditor](https://www.wangeditor.com/) - 简单易用，中文文档完善
- [TinyMCE Vue](https://www.tiny.cloud/docs/tinymce/6/vue-ref/) - 功能全面
