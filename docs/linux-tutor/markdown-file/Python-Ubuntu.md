# Python Ubuntu 安装

## 为 Python 开发环境准备

#### 给 root 用户安装 Python

- 切换用户：`sudo su -`
- 安装 Python 2：`apt-get install python`
- 安装 Python 3：`apt-get install python3`
- 查看版本：
	- `python --version`
	- `python3 --version`

#### 给非 root 用户安装 Anaconda（Python 3 版本）

- 官网下载：<https://www.anaconda.com/download/#linux>
- 进入命令行交互安装：`sh Anaconda3-5.2.0-Linux-x86_64.sh`
