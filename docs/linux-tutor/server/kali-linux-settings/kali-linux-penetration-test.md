- 来源：《Kali Linux 渗透测试的艺术》

## 范围界定

- 筹划工作
- 边界分析

## 信息收集

- 互联网上的公开信息
- 域名注册信息（whois）
- DNS 记录分析
- 路由信息
- 利用搜索引擎搜索目标历史记录
	- Google Hack，根据 Google 的 `inurl` 等高级用法查询一些系统可能存在风险
	- 比如查找链接是：`asp?id=`、`php?id=1` 的链接。asp 和 php 的系统相对比较薄弱。特别是对于 asp 的老系统，建议开发者还是放弃。
