# htop — 交互式进程管理器

`htop` 是 `top` 命令的增强替代品，提供了更直观的交互式界面、彩色显示、鼠标支持和更便捷的进程管理操作。与 `top` 相比，`htop` 支持垂直和水平滚动，可以查看完整的命令行。

## 安装

```bash
# Debian/Ubuntu
sudo apt install htop

# CentOS/RHEL
sudo yum install htop

# macOS
brew install htop
```

## 命令格式

```bash
htop [选项]
```

## 常用参数

| 参数 | 说明 |
|------|------|
| `-d 秒` | 刷新间隔，默认 1 秒 |
| `-u 用户` | 仅显示指定用户的进程 |
| `-p PID1,PID2` | 仅监控指定 PID |
| `-s 列名` | 按指定列排序 |
| `--tree` | 树形显示进程 |
| `-h` | 显示帮助信息 |

### 功能键

| 按键 | 说明 |
|------|------|
| `F1` | 显示帮助信息 |
| `F2` | 进入设置菜单 |
| `F3` | 搜索进程 |
| `F4` | 过滤进程 |
| `F5` | 树形显示切换 |
| `F6` | 选择排序方式 |
| `F7` | 降低优先级（nice +） |
| `F8` | 提高优先级（nice -） |
| `F9` | 杀死进程 |
| `F10` | 退出 |

## 使用示例

### 基本使用

```bash
$ htop
```

### 监控特定用户

```bash
$ htop -u mysql
```

### 树形显示进程

```bash
$ htop --tree
```

### 设置刷新间隔

```bash
$ htop -d 5    # 每 5 秒刷新一次
```

## htop 与 top 对比

| 特性 | top | htop |
|------|-----|------|
| 颜色高亮 | 基本支持 | 全面支持 |
| 鼠标支持 | 不支持 | 支持 |
| 水平滚动 | 不支持 | 支持 |
| 垂直滚动 | 不支持 | 支持 |
| 树形视图 | 支持（V 键） | 支持（F5） |
| 搜索进程 | 不支持 | 支持（F3） |
| 过滤进程 | 不支持 | 支持（F4） |
| 杀死进程 | 交互式（k 键） | 交互式（F9） |
| 配置文件 | `~/.toprc` | `~/.config/htop/htoprc` |
