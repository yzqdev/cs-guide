# Minecraft MOD 推荐

以下是一些实用的 Minecraft MOD，主要基于 Fabric 平台。

## 性能优化类

| MOD 名称 | 功能描述 |
|---------|---------|
| **Lithium** | 全面优化游戏性能，减少卡顿 |
| **Phosphor** | 光照系统优化，提升渲染效率 |
| **Dynamic FPS** | 动态调整 FPS，降低资源占用 |
| **FastBench** | 工作台操作优化 |
| **FastFurnace** | 熔炉烧炼速度优化 |
| **SmoothBoot** | 平滑启动，减少加载时间 |
| **OptiFine** | 高清修复，支持光影和高清材质 |

## 前置依赖类

| MOD 名称 | 功能描述 |
|---------|---------|
| **Fabric API** | Fabric 基础 API |
| **Malilib** | Masa 系列 MOD 的前置 |
| **Mod Menu** | MOD 列表管理界面 |
| **Architectury** | 跨平台架构支持 |
| **OptiFabric** | OptiFine 的 Fabric 兼容层 |

## 功能增强类

| MOD 名称 | 功能描述 |
|---------|---------|
| **REI (Roughly Enough Items)** | 物品管理器，查看配方 |
| **RER (Resourceful Enough Resources)** | 资源管理器 |
| **Inventory Profiles** | R 键整理背包 |
| **VoxelMap** | 小地图 |
| **Light Overlay** | F7 光照显示 |
| **Easier Villager Trading** | 快捷村民交易 |
| **TellMe** | 指令建议 |
| **Item Scroller** | 滚轮滑动物品 |

## 界面显示类

| MOD 名称 | 功能描述 |
|---------|---------|
| **BetterF3** | 改进的 F3 调试界面 |
| **Appleskin** | 显示饥饿值和饱和度 |
| **Shulker Box Tooltip** | 潜影箱预览 |
| **Better Enchanted Books** | 美化附魔书显示 |
| **Advancements Enlarger** | 放大进度界面 |
| **Durability Viewer** | 耐久度显示 |
| **Orderly** | 血量显示 |
| **uBlowFire** | 降低火焰遮挡 |

## 修复补丁类

| MOD 名称 | 功能描述 |
|---------|---------|
| **CustomSkinLoader** | 皮肤补丁，支持外置皮肤 |
| **IMBlocker** | 输入法冲突修复 |
| **RandomPatches** | 各种小修复 |

## 美化类

| MOD 名称 | 功能描述 |
|---------|---------|
| **Slight GUI Modifications** | 细微的界面美化 |
| **Better Mod Button** | 美化 MOD 按钮 |

> **注意**: 以上 MOD 主要适用于 **1.16.x** 版本，使用前请确认版本兼容性。
