# Minecraft 服务器插件推荐

以下是适用于 Spigot/Paper 服务器的常用插件推荐，涵盖核心基础、权限管理、经济系统、商店交易、世界管理、性能优化等各个类别。

> **提示**: 安装前请确认插件版本与你的服务端核心版本兼容。建议使用 [PlugManX](https://www.spigotmc.org/resources/plugmanx.88135/) 进行插件管理。

## 核心基础类

> 服务器的基础支撑插件，几乎是每一个服务器都必须安装的。

| 插件名称 | 功能描述 | 链接 |
|---------|---------|------|
| **Vault** | 经济和权限 API 桥接，几乎所有的经济/权限插件都依赖它 | [SpigotMC](https://www.spigotmc.org/resources/vault.34315/) |
| **PlaceholderAPI** | 变量占位符系统，允许在消息、菜单、计分板中使用动态变量 | [SpigotMC](https://www.spigotmc.org/resources/placeholderapi.6245/) |
| **ProtocolLib** | 协议级别的 API，拦截和发送 Minecraft 网络数据包 | [SpigotMC](https://www.spigotmc.org/resources/protocollib.1997/) |
| **EssentialsX** | 基础功能性插件的合集，包含传送、公告、经济、飞行等 | [SpigotMC](https://www.spigotmc.org/resources/essentialsx.9089/) |

## 权限管理类

| 插件名称 | 功能描述 | 链接 |
|---------|---------|------|
