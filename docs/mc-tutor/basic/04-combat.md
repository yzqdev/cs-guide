# Minecraft 战斗与装备

![钻石剑](https://minecraft.wiki/images/Invicon_Diamond_Sword.png)

## 战斗机制

### 攻击冷却

1.9+ 版本引入了攻击冷却机制：

```text
攻击冷却指示器：武器下方的刻度条
满冷却时攻击 = 最大伤害（暴击）
冷却不足时攻击 = 伤害降低
```

### 暴击

```text
暴击条件：
1. 玩家在下落过程中攻击
2. 非空中/水中状态
3. 攻击冷却满

暴击效果：伤害 × 1.5，★粒子效果
触发方式：跳跃 → 下落 → 攻击
```

### 横扫攻击

```text
剑可以同时攻击多个目标
武器冷却满时有横扫效果
横扫之刃附魔增加横扫伤害
```

## 盾牌使用

```text
盾牌合成 (shield)：
IWI
III
 I
W=木板(oak_planks) I=铁锭(iron_ingot)

使用：右键举盾
功能：格挡正面攻击（减伤）
反噬：成功格挡后敌人会后退
弱点：斧头攻击可使盾牌失效5秒
```

## 药水系统

### 酿造基础

```text
酿造台合成：
  BLB
 CBC
B=烈焰棒 L=岩浆膏 C=圆石

基础材料：
- 粗制的药水：下界疣 + 水瓶
- 红石粉：延长持续时间
- 萤石粉：加强效果等级
- 火药：变成喷溅药水
- 龙息：变成滞留药水
```

## 实战策略

### PVE（对抗怪物）

```text
1. 骷髅射手：举盾靠近，近战解决
2. 苦力怕：剑击退后拉开距离
3. 蜘蛛：原地攻击（蜘蛛需要助跑才能跳）
4. 末影人：不要直视，用水隔离
5. 女巫：优先击杀，喝抗火药水应对中毒
6. 掠夺者：盾牌格挡箭矢
7. 卫道士：斧头伤害高，注意格挡
```

### 下界生存

```text
必备准备：
✅ 下界合金/钻石装备
✅ 抗火药水（对岩浆怪和火焰环境）
✅ 足够的食物
✅ 床位（在下床会爆炸！用重生锚替代）
✅ 金锭（与猪灵交易）

猪灵交易：丢金锭给猪灵，随机回馈物品
猪灵蛮兵：不可交易，主动攻击
```

### 末地战斗

```text
对抗末影龙：
1. 破坏治疗水晶（黑曜石柱顶端）
2. 射箭攻击龙
3. 末影珍珠可传送到末地岛
4. 准备大量箭矢
5. 慢落药水防止摔死
6. 床爆炸对龙有伤害（末地可用）

寻找末地城：
1. 杀死龙后跳入传送门
2. 在末地外岛探索
3. 获得鞘翅和潜影壳
```

## 装备进阶

### 下界合金装备获取

```text
1. 进入下界挖远古残骸（Y=15左右）
2. 用4个远古残骸+4个金锭合成下界合金锭
3. 用锻造台将钻石装备升级
```

### 鞘翅使用

```text
鞘翅获取：末地船中
使用：按跳跃键飞行
加速：烟花火箭（可用）
耐久：经验修补修复
```

### 图腾效果

```text
不死图腾：死亡时复活（给予生命恢复II）
获取：杀死唤魔者
建议：主手或副手持拿
```
