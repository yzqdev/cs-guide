## 核心工具合成

### 基础工具

```text
木镐：     木斧：     木剑：     木锹：
PPP        PP         P          P
 S         S         P          S
 S         S                    S
P=木板 S=木棍
```

### 升级工具

```text
铁镐：     铁斧：     铁剑：     铁锹：
III        II         I          I
 S         S         I          S
 S         S                    S
I=铁锭 S=木棍
```

### 下界合金升级

```text
钻石工具 + 下界合金锭 → 下界合金工具
通过锻造台（Smithing Table）升级
```

## 重要方块合成

```text
熔炉：     漏斗：     活塞：     发射器：
CCC        III        WWW        CCC
C C        I I        CIC        C C
CCC        I I        WRW        CRC
C=圆石  I=铁锭  W=木板  R=红石  C=圆石

附魔台：     铁砧：     酿造台：
B B          III        BLB
DCD         II I        CBC
DCD         III
B=书  D=钻石  C=黑曜石  I=铁块  L=烈焰棒
```

## 矿物块与制品

```text
矿物块合成：9个矿物 → 1个块（压缩存储）
块分解：1个块 → 9个矿物

铁块   ← 9铁锭     钻石块 ← 9钻石
金块   ← 9金锭     绿宝石块 ← 9绿宝石
下界合金块 ← 9下界合金锭
```

## 实用技巧

```text
1. 快速砍树：砍掉最下方的原木后，按住左键向上看
2. 精准采集附魔镐可获取完整矿物块
3. 时运附魔镐增加矿物掉落数量
4. 用脚手架（6竹竿）方便高空建筑
5. 水下用门/栅栏门创造呼吸空间
6. 用楼梯代替台阶，节省材料
7. 铁砧可以修复工具和合并附魔
```
