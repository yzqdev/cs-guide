# Minecraft 红石基础

## 红石基本概念

红石是 Minecraft 中的"电路"系统，可以模拟数字逻辑。

```text
红石信号 = 电流
红石粉 (redstone) = 导线
红石火把 (redstone_torch) = 信号源/非门
中继器 (repeater) = 信号增强/延时
比较器 (comparator) = 信号比较/减法
```

## 基础元件

### 红石粉

```text
放置方式：地面平铺
信号强度：0–15（逐格衰减）
最大传输距离：15格
```

### 红石火把

```text
特性：持续输出15级信号
行为：当附着的方块被激活时熄灭（非门）
```

### 红石中继器

```text
功能：1. 增强信号（到满格15）
      2. 单向传输（防止回流）
      3. 延迟（1-4 tick可调）
```

### 红石比较器

```text
模式1（比较模式）：A ≥ B 输出A，否则0
模式2（减法模式）：输出 A - B

右键切换模式
```

## 逻辑门

### 非门（NOT）

```text
输入 →[红石火把]→ 输出（取反）

输入1 → 输出0
输入0 → 输出1
```

### 与门（AND）

```text
输入A ──┐
         ├→[输出]
输入B ──┘

两个输入都是1时输出1
```

### 或门（OR）

```text
输入A ──┐
         ├→[输出]
输入B ──┘

任意输入为1时输出1
```

### RS 触发器

```text
  ┌─[非门]←┐
  │         │
设置→→→ 输出 ←←←重置
  │         │
  └→[非门]─┘

特点：保持状态
设置=ON → 输出=ON
重置=ON → 输出=OFF
```

## 实用元件

### 活塞 (piston)

```text
普通活塞 (piston)：推动最多12个方块
黏性活塞 (sticky_piston)：推动+拉回方块
```

### 发射器 (dispenser) vs 投掷器 (dropper)

| 元件 | ID | 行为 | 用途 |
|------|----|------|------|
| 发射器 | `dispenser` | 激活时发射物品 | 射箭、喷水、放TNT |
| 投掷器 | `dropper` | 激活时投出物品 | 物品运输 |

### 漏斗 (hopper)

```text
漏斗 (hopper)：收集/传输物品
方向：指向的方向输出物品
特性：每 4 tick 传输1个物品
```

### 侦测器 (observer)

```text
功能：检测方块状态变化
输出：单次脉冲（1 tick）
方向：正面为检测面
```

### 音符盒 (note_block)

```text
用途：播放音效
音高：右键调整（下方方块决定音色）
红石信号触发
```

### 钟 (bell)

```text
功能：被红石激活或右键时发声
范围：32格内的村民回到房屋
```

## 常见红石装置

### 自动农场

```text
[水]          [水]          [水]
  ↓            ↓            ↓
[耕地]→[漏斗]→[箱子]

水流收集作物到漏斗中
红石时钟控制活塞/发射器
```

### 物品分类机

```text
[漏斗] → [比较器]
   ↓
[装物品的漏斗]
   ↓
[箱子]

原理：比较器检测漏斗内物品数量
满45个时输出信号锁定漏斗
```

### 简易陷阱

```text
压力板 → [红石粉] → [活塞/发射器]

或：
绊线钩 + 线 → 绊线 → [TNT/箭]
```

### 密码锁

```text
多个拉杆 → [与门/或门] → [输出]
每个拉杆对应一个二进制位
组合正确时开门
```

### 电梯（黏液块方案）

```text
黏性活塞 → 黏液块 → 上升平台
红石中继器 + 火把 → 时序控制
```

## 进阶技巧

```text
1. 红石火把每1 tick恢复，可做高频脉冲
2. 两个中继器对放可制作循环电路
3. 漏斗矿车 = 移动漏斗
4. 观察者 + 活塞 = 即时信号传输
5. 讲台 + 比较器 = 可调节信号强度
6. 阳光传感器可检测白天/夜晚
7. 陷阱箱被打开时输出信号
8. 红石块既可作为电源也可作为导线
```

## 常用电路图

```text
高频时钟：
[中继器] → [红石粉] → [中继器]
                                          ↑          |
                                          |          |
                                          └──────────┘

脉冲发生器（按一次出一次）：
按钮 → [中继器] → [红石粉] → 输出
```

## 实际案例：简易西瓜/南瓜机

```text
1. 耕地一行，种西瓜/南瓜种子
2. 第一个方块旁放活塞
3. 活塞后放红石粉 + 中继器 + 观察者
4. 观察者检测西瓜长大 → 触发活塞 → 破坏西瓜
5. 水流收集掉落物到箱子
```
