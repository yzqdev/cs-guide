# Minecraft 附魔系统

## 附魔基础

附魔可以增强工具、武器和装备的能力。

```text
附魔方式：
1. 附魔台（Enchantment Table）—— 消耗经验和青金石
2. 铁砧（Anvil）—— 合并附魔书
3. 村民交易 —— 购买附魔书
```

### 附魔台

```text
附魔台合成：
   B B
  DCD
  DCD
B=书 D=钻石 C=黑曜石

书架摆放（最大30级附魔）：
  书架围绕附魔台1格远放置
  共需15个书架
```

## 工具附魔

| 附魔 | 最高等级 | 适用工具 | 效果 |
|------|----------|----------|------|
| **效率** | V | 镐/斧/锹/锄 | 加快挖掘速度 |
| **精准采集** | I | 镐/斧/锹 | 采集方块本身（如矿石） |
| **时运** | III | 镐/斧/锹 | 增加掉落物数量 |
| **耐久** | III | 所有工具 | 减少耐久损耗 |
| **经验修补** | I | 所有工具/装备 | 捡经验球修复耐久 |

```text
效率 V + 下界合金镐 = 瞬间挖掘部分方块
精准采集 vs 时运：二者互斥，不能同时附魔
```

## 附魔书（铁砧合成）

通过村民交易或钓鱼获得的附魔书，可以在铁砧上合并到装备上。

```text
铁砧合成：
III
 I 
III

铁砧功能：
1. 修复工具（消耗经验）
2. 合并附魔（将附魔书附到装备上）
3. 重命名物品
```

## 最佳附魔推荐

### 挖矿装备

```text
镐子：效率 V + 时运 III + 耐久 III + 经验修补
备用镐：效率 V + 精准采集 + 耐久 III + 经验修补
```

### 战斗装备

```text
剑：锋利 V + 击退 II + 火焰附加 II + 耐久 III + 经验修补
弓：力量 V + 冲击 II + 火矢 + 无限（或耐久 III + 经验修补）
弩：穿透 IV + 快速装填 III + 耐久 III
```

### 全套盔甲

```text
头盔：保护 IV + 耐久 III + 水下呼吸 III + 经验修补
胸甲：保护 IV + 耐久 III + 荆棘 III + 经验修补
护腿：保护 IV + 耐久 III + 经验修补
靴子：保护 IV + 耐久 III + 摔落保护 IV + 经验修补
```

## 附魔经验消耗计算

```text
附魔等级消耗 = 附魔等级 + (书架数量 / 2)
最大附魔等级 = 30（需要15个书架）

铁砧消耗：
  第一次附魔：1级
  第二次附魔：2级
  第三次附魔：4级
  第四次附魔：8级
  ...
  每操作一次，消耗翻倍
```

## 实用技巧

```text
1. 先附魔再合并比直接附魔更省经验
2. 用较低等级附魔多次，比一次高等级更划算
3. 钓鱼可以获得附魔书
4. 图书馆管理员村民出售附魔书
5. 下界合金工具可以在铁砧上用下界合金锭修复
6. 经验修补 + 耐久 III = 永不损坏（只要在刷经验）
7. 水下速掘 I + 效率 V = 水下也能快速挖掘
```
