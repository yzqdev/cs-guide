# Minecraft MOD 推荐

以下是一些实用的 Minecraft MOD，主要基于 Fabric 平台。Fabric 是新一代轻量级 MOD 加载器，相比 Forge 启动更快、更新更及时。

> **版本提示**: 以下 MOD 主要适用于 **1.16.x ~ 1.20.x** 版本，使用前请确认版本兼容性。建议搭配 [Mod Menu](https://modrinth.com/mod/modmenu) 管理 MOD 列表。

## 性能优化类

> 让你的 Minecraft 跑得更流畅，低配电脑必备。

| MOD 名称 | 功能描述 | 推荐指数 |
|---------|---------|---------|
| **Sodium** | 下一代渲染优化引擎，大幅提升 FPS（比 OptiFine 性能更好） | ⭐⭐⭐⭐⭐ |
| **Lithium** | 全面优化游戏逻辑运算，减少卡顿 | ⭐⭐⭐⭐⭐ |
| **Phosphor** | 光照系统优化，提升渲染效率 | ⭐⭐⭐⭐ |
| **Starlight** | 重写光照引擎，比 Phosphor 更快 | ⭐⭐⭐⭐⭐ |
| **Dynamic FPS** | 后台时自动降低 FPS，减少资源占用 | ⭐⭐⭐⭐ |
| **FastBench** | 工作台操作优化，降低打开合成表时的卡顿 | ⭐⭐⭐ |
| **FastFurnace** | 熔炉烧炼逻辑优化 | ⭐⭐⭐ |
| **SmoothBoot** | 平滑启动，优化内存使用，减少加载时间 | ⭐⭐⭐⭐ |
| **OptiFine** | 经典高清修复，支持光影和高清材质包 | ⭐⭐⭐⭐ |
| **Entity Culling** | 不渲染被遮挡的实体，提升帧率 | ⭐⭐⭐⭐ |
| **Krypton** | 网络优化，减少心跳包延迟 | ⭐⭐⭐ |

## 前置依赖类

> 很多 MOD 需要这些基础库才能运行。

| MOD 名称 | 功能描述 |
|---------|---------|
| **Fabric API** | Fabric 基础 API，几乎所有 Fabric MOD 的前置 |
| **Fabric Language Kotlin** | Kotlin 语言支持，部分 MOD 需要 |
| **Mod Menu** | MOD 列表管理界面，方便查看和管理已安装 MOD |
| **Cloth Config API** | 配置界面 API，许多 MOD 的设置界面依赖 |
| **Architectury API** | 跨平台架构支持（同时兼容 Fabric 和 Forge） |
| **OptiFabric** | OptiFine 的 Fabric 兼容层 |
| **Malilib** | Masa 系列 MOD（如 Tweakeroo、MiniHUD）的前置 |
| **YetAnotherConfigLib (YACL)** | 现代配置库，部分 MOD 使用 |
| **Catalogue** | 美化 MOD 列表界面，替换 Mod Menu |

## 功能增强类

> 游戏体验大幅提升，用了就回不去。

| MOD 名称 | 功能描述 |
|---------|---------|
| **REI (Roughly Enough Items)** | 物品管理器，查看合成配方和使用方式 |
| **EMI** | REI 的现代替代品，更轻量更快 |
| **JEI (Just Enough Items)** | Forge 版最流行的物品管理器 |
| **AppleSkin** | 显示饥饿值和饱和度，直观了解食物效果 |
| **Inventory Profiles Next** | R 键整理背包、快速转移物品 |
| **Xaero's Minimap** | 小地图，支持路径点和实体雷达 |
| **Xaero's World Map** | Xaero 的全屏大地图 |
| **Light Overlay** | 按 F7 显示光照等级，帮助防止怪物生成 |
| **Jade** | 显示指向的方块/实体信息，Waila 的继任者 |
| **Easier Villager Trading** | 一键交易，快速刷新村民职业 |
| **Item Scroller** | 滚轮滑动物品、快速物品移动 |
| **Tweakeroo** | Masa 系列实用工具，包含假潜行、自动工具切换等 |

## 界面显示类

> 让 HUD 更美、信息更丰富。

| MOD 名称 | 功能描述 |
|---------|---------|
| **BetterF3** | 改进的 F3 调试界面，分类展示信息，可按需隐藏 |
| **MiniHUD** | 信息显示增强，可自定义显示坐标、FPS、时间等 |
| **Shulker Box Tooltip** | 潜影箱内容预览，不用打开就能看到里面的物品 |
| **Better Enchanted Books** | 美化附魔书在物品栏中的显示 |
| **Advancements Enlarger** | 放大成就/进度界面，方便截图 |
| **Durability Viewer** | 显示装备耐久度数值 |
| **Orderly** | 显示生物血量条 |
| **uBlowFire** | 降低火焰遮挡视线效果 |
| **Better Statistics Screen** | 改进的统计信息界面，分类展示 |

## 修复补丁类

> 修复各种烦人的 Bug。

| MOD 名称 | 功能描述 |
|---------|---------|
| **CustomSkinLoader** | 皮肤补丁，支持外置皮肤系统（如 LittleSkin） |
| **IMBlocker** | 输入法冲突修复，解决中文输入法卡键问题 |
| **RandomPatches** | 各种小修复的合集 |

## 美化类

> 让游戏的每个角落都赏心悦目。

| MOD 名称 | 功能描述 |
|---------|---------|
| **Slight GUI Modifications** | 细微的界面美化调整 |
| **Better Mod Button** | 美化主界面的 MOD 按钮 |
| **Continuity** | 连接材质支持（类似 OptiFine 的连接纹理功能） |
| **Indium** | Sodium 的渲染 API 扩展，支持更多渲染 MOD |
| **Iris** | 光影加载器，完全开源，与 Sodium 兼容
