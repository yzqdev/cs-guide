# Minecraft 探索与结构

## 主世界结构

### 村庄

```text
生物群系：平原、沙漠、热带草原、针叶林、雪原

村民类型：随村庄群系变化
资源：食物、绿宝石交易、床、箱子

防御：铁傀儡保护
```

### 废弃矿井

```text
生成：地下任意位置
特征：木制支撑柱、铁轨、矿车
危险：洞穴蜘蛛刷怪笼
资源：铁轨、红石、矿物
```

### 要塞

```text
生成：地下随机分布
特征：石砖结构、有传送门房间
资源：末地传送门、箱子、图书馆
定位：用末影之眼（烈焰粉+末影珍珠）
```

### 林地府邸

```text
生成：黑森林
特征：大型木石建筑
生物：卫道士、唤魔者
资源：不死图腾、钻石、金苹果
```

### 海底遗迹

```text
生成：深海生物群系
守卫者生成
资源：海绵、海晶石、金块
```

### 沙漠神殿

```text
生成：沙漠
特征：金字塔形
机关：蓝色陶瓦（压力板+TNT）
资源：钻石、铁、绿宝石
```

### 丛林神庙

```text
生成：丛林
特征：机关解锁
资源：弓箭、骨头、红石
```

### 掠夺者前哨站

```text
生成：任何可生成村庄的群系
特征：瞭望塔
生物：掠夺者、掠食兽
资源：箱子、不祥旗帜
```

### 沼泽小屋

```text
生成：沼泽
生物：女巫
资源：药水、红石、蜘蛛网
```

### 沉船

```text
生成：海洋
资源：藏宝图、食物、铁
引导：使用藏宝图找到埋藏的宝藏
```

## 探索技巧

### 生存工具包

```text
探索必备品：
✅ 满附魔钻石/下界合金装备
✅ 足够的食物（金胡萝卜最佳）
✅ 火把（或夜视药水）
✅ 床（设置重生点）
✅ 末影珍珠（逃离危险）
✅ 水桶（高空救急/跨岩浆）
✅ 盾牌（格挡箭矢）
✅ 弓+箭（远程攻击）
✅ 工作台+合成材料（应急）
✅ 指南针/地图（不迷路）
✅ 命名牌（标记坐标）
```

### 定位方法

```text
定位结构指令：
/locate structure minecraft:village
/locate structure minecraft:fortress
/locate biome minecraft:plains
/locate structure #village

1.21+ 版本：
/ locate structure 后跟结构类型
探险家地图：与制图师村民交易
```

### 常见陷阱

```text
1. 不要在陌生矿道跑（小心悬崖）
2. 不要在高处直接跳（水桶救急）
3. 不要在地狱使用床（会爆炸）
4. 不要直视末影人
5. 不要靠近苦力怕（盾牌格挡）
6. 不要在水里穿钻石靴（沉底）
```

### 资源收集优先级

```text
第一天：
木材 > 食物 > 庇护所 > 床

前期：
铁 > 煤炭 > 食物 > 更多木材

中期：
钻石 > 下界合金 > 附魔材料

后期：
鞘翅 > 潜影壳 > 龙头（装饰）
```
