# Minecraft 种植与养殖

## 种植系统

### 基础种植

```text
种植步骤：
1. 用锄头右键耕地
2. 水源在4格范围内（湿润耕地）
3. 播种
4. 等待生长（光照和水分充足）
5. 收获
```

### 骨粉机制

```text
骨粉 = 骨粉催熟
来源：骷髅掉落 → 合成骨粉

效果：
- 可催熟大部分作物
- 可催树苗生长
- 可催花和草
- 对地狱疣无效
```

### 自动农场设计

```text
基础电路：
观察者 检测作物成熟
→ 触发红石信号
→ 激活活塞/发射器收获
→ 水流收集到漏斗

进阶：
漏斗矿车 收集掉落物
箱子车 站点卸货
时运附魔锄 增加作物掉落
```

## 养殖系统

### 动物繁殖

```text
繁殖条件：
1. 两只同种动物
2. 手持对应食物
3. 右键喂食
4. 5分钟冷却

幼崽生长：20分钟（可喂食加速）
```

## 村民繁殖与交易

### 村民繁殖条件

```text
1. 至少有两名村民
2. 有效床（每个村民一个床）
3. 有工作站
4. 有食物（面包/胡萝卜/马铃薯/甜菜根）
5. 村民愿意繁殖（有食物且2分钟内交易过）
```

## 蜜蜂养殖

```text
蜜蜂养殖步骤：
1. 找到蜂巢（盛开的杜鹃树附近）
2. 用附有精准采集的工具采集蜂巢
3. 放置蜂巢到农场
4. 在旁边种花
5. 蜜蜂采蜜 → 蜂巢蜜汁满 → 用玻璃瓶收集蜂蜜
6. 用剪刀收集蜜脾

注意：采蜜时蜂巢会激怒蜜蜂
方案A：下方放营火（不激怒蜜蜂）
方案B：用发射器采蜜
```

## 实用小技巧

```text
1. 耕地湿润范围：水源4格内
2. 耕地会因踩踏变回泥土（跳跃）
3. 半砖和楼梯上方不会踩踏耕地
4. 火把提供光照促进夜间生长
5. 海龟蛋吸引僵尸和骷髅
6. 用命名牌命名动物防止消失
7. 拴绳可以捆绑和移动动物
8. 动物会跟随手持食物的玩家
```
