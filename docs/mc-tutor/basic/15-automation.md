# 自动化农场与刷资源

> 从手动到全自动，覆盖 Minecraft 中最实用的自动农场和资源刷取方案。

![漏斗](https://minecraft.wiki/images/Invicon_Hopper.png) ![活塞](https://minecraft.wiki/images/Invicon_Piston.png) ![观察者](https://minecraft.wiki/images/Invicon_Observer.png) ![中继器](https://minecraft.wiki/images/Invicon_Redstone_Repeater.png) ![比较器](https://minecraft.wiki/images/Invicon_Redstone_Comparator.png)

---

## 基础原理

### 自动化的核心要素

```text
1. 生成/来源     → 怪物、作物、村民等
2. 收集/运输     → 水流、漏斗、气泡柱
3. 处理/击杀     → 掉落伤害、火焰、改名
4. 存储          → 箱子 + 漏斗链
```

### 常用收集机制

| 方式 | 速度 | 优点 | 缺点 |
|------|------|------|------|
| **水流** | 极快 | 无消耗、可定向 | 需要等高差 |
| **漏斗 (hopper)** | 2.5 物品/秒 | 精准收集、可分类 | 较贵（5 铁） |
| **气泡柱 (bubble_column)** | 极快 | 垂直运输 | 需要海草(kelp) + 水源 |
| **漏斗矿车 (hopper_minecart)** | 快 | 可穿越长距离 | 铁轨成本高 |

## 作物自动农场

### 简易水流农田

```text
# 布局（8×8 农场）
[水源]                          [收集]
  ↓                               ↓
 田 田 田 田 田 田 田 田    → 水流
 田 田 田 田 田 田 田 田    → 水流
 田 田 田 田 田 田 田 田    → 水流
 ...
                              ↓
                          [漏斗+箱子]

# 工作方式
1. 手动或村民种地
2. 成熟后放水 → 冲走作物
3. 水流汇集到收集点
4. 漏斗收集进箱子

# 半自动（村民辅助）
- 农民村民会种地 + 收菜
- 村民会自动丢出多余食物
- 漏斗收集村民丢出的食物
- 完全不需要玩家动手！
```

### 甘蔗农场

```text
# 自动甘蔗机

   [观察者(observer)]          [活塞(piston)]
    │                  │
    ├──→ 检测甘蔗(sugar_cane) → 推动 → 收集
    │
   [甘蔗] ← 旁边放水
    │
   [沙子(sand)/泥土(dirt)]

# 每棵甘蔗放一个观察者 + 活塞
# 甘蔗长到 3 格高 → 观察者检测 → 活塞打掉上面 2 格 → 掉落物被水冲走

# 优点
- 完全自动（不需要玩家）
- 甘蔗可用来做纸（附魔 + 烟火）
```

### 竹子农场

同甘蔗农场原理，使用观察者 + 活塞检测竹子生长。

```text
# 特点
- 竹子是所有作物中生长最快的
- 很好的燃料（烧炼材料）
- 制作脚手架（建筑利器）
```

## 怪物农场

### 简易刷怪塔

```text
# 原理
怪物在黑暗处生成 → 掉落到击杀平台 → 收集掉落物

# 基本结构
          [刷怪层]
     ┌─────┬─┬─────┐
     │ █████ █████ │  ← 暗室（亮度 0）
     │ █   █ █   █ │
     │ █   █ █   █ │  ← 2 格高空间
     │ █   █ █   █ │
     └─────┴─┴─────┘
           │
           │ 水流 + 掉落
           │
     ┌─────┴─────┐
     │  击杀区    │ ← 玩家站这里（或岩浆块）
     └─────┬─────┘
           │ 漏斗
         [箱子]
```

### 怪物农场的选址

```text
# 最佳位置：高空（Y=200+）
- 周围洞穴不会生成怪物（全部在刷怪塔生成）
- 效率最高

# 其次：海上
- 周围没有陆地 → 不会生成地面怪物
- 需要平台

# 最差：地下
- 周围大量洞穴 → 怪物生成被分散
- 需要大面积挖空
```

### 常用怪物农场

| 农场类型 | 主要产出 | 难度 | 说明 |
|----------|---------|------|------|
| **通用刷怪塔** | 火药(gunpowder)、骨头(bone)、线(string)、箭(arrow) | ⭐⭐ | 基础且实用 |
| **苦力怕农场** | 大量火药 | ⭐⭐⭐ | 苦力怕(creeper)是唯一掉落火药的怪物 |
| **骷髅农场** | 骨头、箭 | ⭐⭐ | 用刷怪笼(spawner)改造 |
| **蜘蛛农场** | 线 | ⭐⭐ | 用刷怪笼改造 |
| **烈焰人农场** | 烈焰棒(blaze_rod) | ⭐⭐⭐ | 下界要塞刷怪笼改造 |
| **凋灵骷髅农场** | 下界之星(nether_star) | ⭐⭐⭐⭐ | 获得信标(beacon)必需 |
| **猪灵金农场** | 金粒(gold_nugget)、金锭(gold_ingot) | ⭐⭐⭐ | 下界堡垒，大量经验 |

### 经验农场

利用刷怪笼（地下的刷怪笼房间）改造：

```text
# 改造步骤
1. 找到刷怪笼（洞穴蜘蛛/普通蜘蛛/僵尸/骷髅）
2. 在刷怪笼周围点亮所有洞穴（火把）
3. 建造掉落通道（24 格高 → 摔到 1 心）
4. 玩家在底部击打怪物获得经验

# 产出
高级经验农场：30 级 → 1-2 分钟（满级附魔速度）
```

## 村民自动化农场

### 自动胡萝卜/马铃薯农场

```text
# 只需要：
1. 农民村民 × 1
2. 农田 × 1
3. 漏斗 × 1（或水流收集）

# 农民的工作流程
1. 种地 → 收获 → 丢给其他村民
2. 其他村民捡起 → 食物槽位满 → 丢地上
3. 漏斗收集地上的食物
4. 或水流冲到收集点

# 产出效率
一个农民：约 2 组/小时
```

### 自动烤鸡机

```text
# 原理
1. 鸡蛋 → 投掷器 → 小鸡 → 长大
2. 成年鸡 → 火焰/岩浆杀死 → 熟鸡肉

# 简化版（半自动）
1. 在箱子放鸡蛋
2. 投掷器朝一个 1×1 空间投鸡蛋
3. 小鸡长大 → 掉入岩浆块 → 熟鸡肉
4. 漏斗收集
```

## 树场

### 手动树场

种一大片树苗，用斧头砍——适合早期。

```text
# 高效率种植
- 间隔 4×4 种一棵（大树冠）
- 用骨粉催熟
- 用带效率 V 的下界合金斧

# 树苗搭配
橡树：产出苹果
云杉：最直、成长快
从林木：4 格高树苗 → 巨树
```

### 自动树场（简单版）

```text
# 需要
1. 观察者检测树生长
2. 活塞/TNT 破坏树木
3. 水流收集掉落物

# 注意
- 自动树场结构较复杂
- 需要配合 TNT 复制
- 适合云杉（单一高度）
```

## 物品分类系统

### 简易分类机

```text
# 核心原理：漏斗 + 比较器 + 红石火把

           [漏斗] ← 物品流入
            │
           [漏斗] ← 分类漏斗（内有 42 个过滤物品）
            │
    ┌───────┴───────┐
    │ [比较器] → [红石火把] → 锁定上方漏斗
    │ [箱子]        │
    └───────────────┘

# 工作原理
1. 每个分类漏斗放 42 个同类物品
2. 第 43 个物品触发比较器信号
3. 信号激活红石火把 → 锁定漏斗
4. 其他物品流向下一个分类槽

# 分类容量
每个分类单元 = 1 种物品
N 个单元 = N 种物品
```

### 实用分类方案

```text
# 推荐分类组
第一组：常用建筑材料（石头、木头、泥土）
第二组：矿石与金属（铁、金、钻石、下界合金）
第三组：食物（面包、肉、蔬菜）
第四组：怪物掉落物（火药、骨头、线、箭）
第五组：红石与工具（红石、活塞、漏斗）
第六组：杂项（末影珍珠、烈焰粉、书）

# 在漏斗入口前放一个岩浆块（销毁不可堆叠物）
# 避免垃圾堵塞分类系统
```
