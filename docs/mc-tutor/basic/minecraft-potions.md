# 药水酿造指南

> 从酿造台搭建到效果药水，全面掌握 Minecraft 的药水酿造系统。

---

## 酿造基础

### 酿造台

```text
# 合成
  B      B = 烈焰棒
  S      S = 圆石（3 个）
 RSSR
  S

# 放置
酿造台需要烈焰粉作为燃料（1 个烈焰粉 = 20 次酿造）
```

### 主要材料

| 材料 | 来源 | 用途 |
|------|------|------|
| **烈焰粉** | 烈焰棒合成 | 酿造燃料 |
| **水瓶** | 玻璃瓶装水 | 所有药水的基础 |
| **下界疣** | 下界要塞 | 将水瓶酿造成粗制药水（基础中的基础） |
| **红石粉** | 挖矿（Y≤15） | 延长持续时间 |
| **萤石粉** | 下界采集萤石 | 提升效果等级 |
| **火药** | 苦力怕/宝箱 | 制作喷溅药水 |
| **龙息** | 末影龙的龙息攻击 | 制作滞留药水 |
| **发酵蛛眼** | 蛛眼+糖+棕色蘑菇合成 | 将正面效果变负面 |

---

## 酿造流程概览

```text
水瓶
  │
  ├── 下界疣 → 粗制药水（基础药水）
  │              │
  │              ├── 加效果材料 → 效果药水（3:00）
  │              │                │
  │              │                ├── +红石 → 延长版（8:00）
  │              │                ├── +萤石 → 强化版（II级）
  │              │                └── +火药 → 喷溅药水
  │              │                                    │
  │              │                                    └── +龙息 → 滞留药水
  │              │
  │              └── 发酵蛛眼 → 负面效果药水
  │
  ├── 直接加材料 → 特定药水（夜视/隐身/火抗等）
  │
  └── 发酵蛛眼 → 虚弱药水（虚弱直接用水瓶+发酵蛛眼）
```

## 红色 vs 萤石：效果对比

| 特性 | 红石粉 | 萤石粉 |
|------|--------|--------|
| 效果 | 延长持续时间 | 提升效果等级 |
| 速度 I | 3:00 → 8:00 | 速度 I → 速度 II（1:30） |
| 力量 I | 3:00 → 8:00 | 力量 I → 力量 II（1:30） |
| 生命恢复 I | 0:45 → 2:00 | 生命恢复 I → 生命恢复 II（0:22） |
| 注意 | 不能叠加红石+萤石 | 先加萤石再红石无效 |

> 先加红石延长，再加萤石 → 有效。
> 先加萤石强化，再加红石 → **无效**。酿造顺序很重要！

## 药水箭

将箭与滞留药水在酿造台中合成，每瓶滞留药水可制作 **8 支药水箭**。

### 常用药水箭

| 药水箭种类 | 效果 | 推荐场合 |
|------------|------|----------|
| **缓慢箭** | 减缓敌人追击 | PVP、跑路 |
| **伤害箭** | 即时伤害（瞬间伤害 II） | PVP、打 BOSS |
| **中毒箭** | 持续中毒掉血 | 消耗战 |
| **虚弱箭** | 降低敌人攻击力 | BOSS 战 |
| **迟缓箭** | 降低挖掘速度（PVE 无用） | PVP 干扰 |
| **隐身箭** | 被射中者隐身（PVE 无用） | 娱乐 |

## 酿造实用技巧

### 最大化药水产出

```text
# 配方：1 个材料 + 3 瓶粗制药水 → 3 瓶效果药水
# 酿造台一次放 3 瓶最划算

# 材料效率
- 1 烈焰棒 = 2 烈焰粉
- 1 烈焰粉 = 20 次酿造
- 1 次酿造 = 3 瓶药水
- 1 烈焰粉 = 60 瓶药水！
```

### 酿造顺序

```text
1. 水瓶 + 下界疣       → 粗制药水 3 瓶
2. 粗制药水 + 效果材料  → 效果药水 3 瓶
3. 效果药水 + 红石粉    → 延长版（8:00）
   -或-
   效果药水 + 萤石粉    → 强化版（II 级）
4. 效果药水 + 火药      → 喷溅药水
5. 喷溅药水 + 龙息      → 滞留药水
```

### 实用药水清单

```text
# 下界探索必备
抗火药水 × 3（8:00）     # 免疫岩浆和火焰
夜视药水 × 2（8:00）     # 看清下界地形

# 矿工标配

# 战斗配置

瞬间治疗药水 II × 5     # 应急回血（喷溅型可扔队友）

# 末地决战
缓降药水 × 3             # 摔落保护
力量药水 II × 3          # 输出
```

## 信标与效果

信标可以为附近玩家提供持续的状态效果，无需消耗药水。

```text
# 信标激活
1. 建造信标金字塔（铁锭/金锭/钻石/绿宝石块搭建）
2. 金字塔 3×3 → 5×5 → 7×7 → 9×9（层数越高效果越强）
3. 右键信标选择主效果
4. 满级金字塔额外开启副效果（生命恢复）

# 信标可选效果
- 速度 I / II
- 急迫 I / II
- 抗性提升 I / II
- 跳跃提升 I / II
- 力量 I / II
- 生命恢复（副效果）
```
