# Minecraft 游戏基础

## 游戏模式

| 模式 | 说明 | 适合场景 |
|------|------|----------|
| **生存模式** | 收集资源、合成工具、对抗怪物、维持饱食度 | 核心玩法体验 |
| **创造模式** | 无限资源、飞行能力、无敌状态 | 建筑、红石实验 |
| **冒险模式** | 不能破坏/放置方块（除非有对应工具） | 自定义地图 |
| ** spectator模式** | 穿墙、透视、附身生物视角 | 观察、录制 |

```text
切换模式指令：
/gamemode survival       # 生存
/gamemode creative       # 创造
/gamemode adventure      # 冒险
/gamemode spectator      # 旁观
# 简写
/gamemode 0              # 生存
/gamemode 1              # 创造
/gamemode 2              # 冒险
/gamemode 3              # 旁观
```

## 难度等级

| 难度 | 怪物伤害 | 饥饿 | 其他 |
|------|----------|------|------|
| **和平** | 无怪物生成 | 不降 | 自动回血 |
| **简单** | 伤害低 | 降较慢 | 僵尸不会破门 |
| **普通** | 标准伤害 | 标准 | 默认体验 |
| **困难** | 伤害高 | 降得快 | 僵尸破门、饥饿可致死 |

```text
/difficulty peaceful    # 和平
/difficulty easy        # 简单
/difficulty normal      # 普通
/difficulty hard        # 困难
```

## 世界类型

| 类型 | 特点 |
|------|------|
| **默认** | 标准生成，各种生物群系 |
| **超平坦** | 完全平坦，适合建筑/红石实验 |
| **巨型生物群系** | 生物群系放大，真实感强 |
| **放大化** | 地形起伏夸张，高山深谷 |
| **单一生物群系** | 全图同一种群系 |
| **洞穴** | 主世界被洞穴取代（1.18+） |
| **调试模式** | 显示所有方块状态 |

## 坐标系统

Minecraft 使用三维坐标 (X, Y, Z)：

- **X** — 东西方向（东为正）
- **Y** — 高度（0=基岩层，63=海平面）
- **Z** — 南北方向（南为正）

```text
显示坐标：F3 或 /tp ~ ~ ~
设置显示坐标：/gamerule showCoordinates true
```

### 重要高度

| 高度 | 说明 |
|------|------|
| Y=-64 | 世界底部（1.18+） |
| Y=0 | 基岩层顶部 |
| Y=11 | 传统挖钻石最佳层数 |
| Y=63 | 海平面 |
| Y=255 | 世界顶部（1.18+） |
| Y=319 | 建筑上限（1.18+） |

## 时间系统

```text
/day    设置为白天
/night  设置为夜晚
/time set day     # 1000 tick
/time set night   # 13000 tick
/time set 0       # 黎明
/time add 1000    # 增加时间
```

### 游戏刻（Tick）

- 1 秒 = 20 game tick
- 1 天 = 24000 tick = 20 分钟
- 白天: 0–12000 tick
- 日落: 12000–13000 tick
- 夜晚: 13000–23000 tick
- 日出: 23000–24000 tick

## 初始生存流程

### 第一天——建立据点

```text
1. 砍树 → 获得原木
2. 制作工作台：4个木板
3. 制作木镐 → 挖圆石 → 石镐
4. 寻找合适地点（靠山/靠水）
5. 搭建临时庇护所
6. 制作床（3个羊毛 + 3个木板）
7. 制作火把（木棍 + 木炭）
```

### 前期目标

```text
✅ 安全庇护所
✅ 床（设置重生点）
✅ 基本工具（石镐、石斧、石剑）
✅ 火把（防止怪物生成）
✅ 食物来源（杀动物、种小麦）
✅ 熔炉（烧矿、烹饪）
✅ 箱子（存放物品）
```

### 矿物优先级

| 矿物 | ID | 用途 | 推荐层数 | 工具要求 |
|------|----|------|----------|----------|
| 煤矿 | `coal_ore` / `coal` | 火把、燃料 | 任意 | 木镐+ |
| 铁矿 | `iron_ore` / `iron_ingot` | 工具、装备 | Y=15 | 石镐+ |
| 金矿 | `gold_ore` / `gold_ingot` | 下界合金、动力铁轨 | Y=-16 | 铁镐+ |
| 钻石 | `diamond_ore` / `diamond` | 顶级工具、装备 | Y=-59 | 铁镐+ |
| 红石 | `redstone_ore` / `redstone` | 红石电路 | Y=-59 | 铁镐+ |
| 下界合金 | `ancient_debris` / `netherite_ingot` | 升级钻石装备 | 下界 | 钻石镐 |

## 生物群系（Biome）分类

| 类型 | 群系(ID) | 特点 |
|------|---------|------|
| **森林** | 森林(forest)、白桦林(birch_forest)、黑森林(dark_forest) | 木材丰富，怪物较少 |
| **平原** | 平原(plains)、向日葵平原(sunflower_plains) | 视野开阔，动物多 |
| **沙漠** | 沙漠(desert) | 沙子、仙人掌、神殿 |
| **山地** | 风袭丘陵(windswept_hills)、尖峭山峰(jagged_peaks) | 大量露天矿物 |
| **水域** | 海洋(ocean)、河流(river) | 水下遗迹 |
| **洞穴** | 繁茂洞穴(lush_caves)、溶洞(dripstone_caves) | 独特植被、发光浆果 |
| **下界** | 诡异森林(warped_forest)、玄武岩三角洲(basalt_deltas) | 独特资源 |
| **末地** | 末地(the_end)、末地外岛(end_barrens) | 终界龙、紫颂果(chorus_fruit) |
