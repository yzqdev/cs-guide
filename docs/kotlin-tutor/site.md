# Kotlin 相关库与资源

> Kotlin 生态日益丰富，以下是一些常用的库和框架。

## 服务端框架

| 库/框架 | 说明 | 链接 |
|---------|------|------|
| **Ktor** | JetBrains 出品的异步服务端框架 | <https://ktor.io/> |
| **Ktorm** | 轻量级 ORM 框架，纯 Kotlin 编写 | <https://www.ktorm.org/> |
| **Spring Boot + Kotlin** | Spring 官方支持 Kotlin | <https://spring.io/guides/tutorials/spring-boot-kotlin/> |
| **Exposed** | JetBrains 出品的 SQL ORM | <https://github.com/JetBrains/Exposed> |
| **Http4k** | 函数式 HTTP 框架 | <https://www.http4k.org/> |

## 移动端

| 库/框架 | 说明 |
|---------|------|
| **Jetpack Compose** | Android 官方声明式 UI 框架 |
| **Compose Multiplatform** | 跨平台 Compose（Android + Desktop + Web） |
| **Anko** | 已弃用，但仍是 Kotlin Android 开发的先驱 |

## 工具库

| 库 | 说明 | 链接 |
|----|------|------|
| **kotlinx.coroutines** | 协程库，异步编程核心 | <https://github.com/Kotlin/kotlinx.coroutines> |
| **kotlinx.serialization** | 序列化库（JSON/ProtoBuf/CBOR） | <https://github.com/Kotlin/kotlinx.serialization> |
| **Arrow** | 函数式编程库，提供 Either、Option 等类型 | <https://arrow-kt.io/> |
| **MockK** | Kotlin 原生 Mock 测试框架 | <https://mockk.io/> |
| **Kotest** | Kotlin 测试框架 | <https://kotest.io/> |
