Viper 使用注意

解析 yaml 或 json 时，如果出现值为空，通常是 tag 写错了。Viper 内部使用 mapstructure 标签，而非 yaml 或 json：

type Config struct {
    QiniuAccessKey string `yaml:"qiniu_accesskey" mapstructure:"qiniu_accesskey"`
}

关于间接依赖
