Windows 快捷键大全

掌握这些快捷键，让你的 Windows 操作效率翻倍。

通用快捷键

快捷键功能
Ctrl + C复制
Ctrl + X剪切
Ctrl + V粘贴
Ctrl + Z撤销
Ctrl + Y重做
Ctrl + A全选
Ctrl + S保存
Ctrl + F查找
Ctrl + H替换
Ctrl + P打印
Ctrl + Shift + Esc打开任务管理器
Alt + F4关闭当前窗口
Alt + Tab切换窗口
Alt + Space打开窗口菜单（移动/大小/最小化/关闭）
F2重命名
F5刷新
Delete删除到回收站
Shift + Delete永久删除（跳过回收站）

任务管理器快捷键

Alt + E结束任务
Alt + N打开新任务（运行新程序）

命令行快捷键（CMD / PowerShell）

快捷键功能
↑ / ↓浏览命令历史
Tab自动补全路径/文件名
Ctrl + C中断当前命令
Ctrl + A选中整行
Ctrl + ←/→按单词移动光标
Ctrl + Backspace删除前一个单词
F7显示命令历史弹窗
Esc清空当前行

剪贴板增强

Win10/11 内置了剪贴板历史功能：

  1. 按 Win + V 打开剪贴板历史
  2. 可以查看最近复制的多条内容（文本、图片）
  3. 点击任意条目即可粘贴
  4. 可以固定常用条目

需要在 设置 → 系统 → 剪贴板 中开启"剪贴板历史记录"。

虚拟桌面（多桌面）

虚拟桌面适合将不同任务分组：

  • Win + Ctrl + D — 新建桌面
  • Win + Ctrl + ←/→ — 切换桌面
  • Win + Ctrl + F4 — 关闭当前桌面
  • Win + Tab — 查看所有桌面和窗口

使用场景：

  • 桌面1：工作（VS Code + 浏览器 + 终端）
  • 桌面2：聊天通讯（微信 + 钉钉）
  • 桌面3：娱乐（视频 / 音乐）
